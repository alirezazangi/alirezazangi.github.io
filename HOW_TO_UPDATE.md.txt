# راهنمای آپدیت کردن برنامه

این راهنما به شما نشان می‌دهد که چگونه پس از اعمال تغییرات در کد (Rust یا JavaScript)، یک نسخه جدید از برنامه را برای پلتفرم‌های مختلف منتشر کنید تا کاربران آن را به صورت خودکار دریافت کنند.

## فرآیند گام به گام

**مرحله ۱: اعمال تغییرات و افزایش شماره نسخه**
1.  تغییرات مورد نظر خود را در کد اعمال کنید.
2.  فایل `src-tauri/tauri.conf.json` را باز کرده و شماره `version` را افزایش دهید (مثلاً از `1.0.1` به `1.0.2`). این کار **بسیار مهم** است.

**مرحله ۲: ساختن نسخه جدید (Build)**
برای هر سیستم‌عاملی که می‌خواهید پشتیبانی کنید، باید روی همان سیستم‌عامل (یا با استفاده از ابزارهای CI/CD) دستور build را اجرا کنید.

در ترمینال پوشه اصلی پروژه (`New Folder`)، دستور زیر را اجرا کنید:
```bash
npm run tauri build -- --release
```
این دستور فایل‌های نصبی را در مسیر `src-tauri/target/release/bundle` می‌سازد. برای هر پلتفرم، یک فایل فشرده (مثلاً `.msi.zip` برای ویندوز یا `.app.tar.gz` برای مک) و یک فایل امضای دیجیتال (`.sig`) ایجاد می‌شود.

**مرحله ۳: آپلود فایل‌های نصبی روی سرور**
تمام فایل‌های فشرده‌ی ساخته شده در مرحله قبل را روی سرور خود (`alirezazangi.ir`) آپلود کنید.

**مرحله ۴: به‌روزرسانی فایل `updates.json`**
فایل `alirezazangi/updates.json` را باز کنید. این فایل به شما اجازه می‌دهد برای هر پلتفرم، آپدیت مخصوص به آن را مشخص کنید.

1.  مقدار `version` اصلی را به شماره نسخه جدید تغییر دهید.
2.  برای هر پلتفرم، بخش مربوطه را با اطلاعات صحیح پر کنید:
    *   **`signature`**: محتوای کامل فایل `.sig` مربوط به آن پلتفرم را کپی و اینجا جای‌گذاری کنید.
    *   **`url`**: آدرس کامل فایل فشرده آپلود شده روی سرور را وارد کنید.

**کلیدهای پلتفرم:**
-   `windows-x86_64`: برای ویندوز ۶۴ بیتی
-   `darwin-x86_64`: برای مک با پردازنده‌های اینتل
-   `darwin-aarch64`: برای مک با پردازنده‌های اپل سیلیکون (M1, M2, ...)
-   `linux-x86_64`: برای لینوکس ۶۴ بیتی

**مثال برای `updates.json` به‌روز شده:**
```json
{
  "version": "1.0.2",
  "notes": "توضیحات تغییرات این نسخه.",
  "mandatory": true,
  "platforms": {
    "windows-x86_64": {
      "signature": "محتوای کامل فایل .sig ویندوز...",
      "url": "https://alirezazangi.ir/Tahlil-Saham_1.0.2_x64.msi.zip"
    },
    "darwin-aarch64": {
      "signature": "محتوای کامل فایل .sig مک (ARM)...",
      "url": "https://alirezazangi.ir/Tahlil-Saham_1.0.2_aarch64.app.tar.gz"
    }
  }
}
```

**مرحله ۵: آپلود `updates.json`**
فایل `updates.json` ویرایش شده را روی سرور خود در آدرس ریشه (`https://alirezazangi.ir/updates.json`) آپلود کنید.

تمام شد! اکنون برنامه به صورت هوشمند پلتفرم کاربر را تشخیص داده و فقط آپدیت مربوط به همان سیستم‌عامل را دانلود می‌کند.
