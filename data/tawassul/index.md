راهنمای مطالعه امتحان - فصل ۳: پلی‌فسفازن‌ها
📚 مقدمه - چرا پلی‌فسفازن‌ها مهم هستند؟

پلی‌فسفازن‌ها بزرگترین خانواده از پلیمرهای معدنی هستند. تا الان بیش از ۷۰۰ نوع از این پلیمرها ساخته شده که خواص فیزیکی و شیمیایی متنوعی دارند.
ساختار کلی:

    R         R         R
    |         |         |
—N═P—N═P—N═P—N═P—
    |         |         |
    R         R         R
```

- **زنجیره اصلی**: اتم‌های فسفر (P) و نیتروژن (N) به صورت متناوب
- **گروه‌های جانبی (R)**: می‌توانند آلی، فلزی-آلی یا معدنی باشند
- **طول زنجیر**: معمولاً ۱۰۰ تا ۱۵٬۰۰۰ واحد تکراری
- **وزن مولکولی**: ۲ تا ۱۰ میلیون

---

## 🔑 نکات کلیدی برای امتحان

### 1️⃣ **تاریخچه (خیلی مهم!)**

**۱۸۳۴**: لیبیگ و روز اولین ترکیبات را گزارش کردند
- واکنش PCl₅ با آمونیاک → N₃P₃Cl₆

**۱۸۹۷**: استوکس 
- ساختار حلقوی N₃P₃Cl₆ را پیشنهاد داد
- اولین کسی که "لاستیک معدنی" ساخت
- با گرم کردن حلقه‌ها → ماده لاستیکی
- این ماده در حلال حل نمی‌شد (چون cross-linked بود)

**۱۹۶۵-۱۹۶۰**: آلکوک و همکاران 🌟
- **نقطه عطف**: ساخت پلی(دی‌کلروفسفازن) بدون cross-link
- این پلیمر در حلال حل می‌شد!
- می‌توانستند کلرها را با گروه‌های آلی جایگزین کنند
- اولین پلی‌فسفازن پایدار: `[NP(OCH₂CF₃)₂]ₙ`

---

### 2️⃣ **روش‌های سنتز (برای امتحان حتماً بخوان!)**

#### 🔵 **روش اصلی: Ring-Opening + جایگزینی**

**مرحله ۱**: پلیمریزاسیون ring-opening
```
(NPCl₂)₃ --[گرما 210-250°C]--> (NPCl₂)ₙ
(حلقه)                          (زنجیر)
```

**نکته مهم**: 
- باید قبل از ۷۰٪ تبدیل متوقف شود
- وگرنه cross-linking می‌شود و حل نمی‌شود!

**مرحله ۲**: جایگزینی ماکرومولکولی
```
(NPCl₂)ₙ + NaOR → [NP(OR)₂]ₙ + NaCl↓
```

**مثال‌های مهم**:
- `+ NaOCH₂CF₃ → [NP(OCH₂CF₃)₂]ₙ` (اولین پلیمر پایدار)
- `+ C₂H₅NH₂ → [NP(NHC₂H₅)₂]ₙ`
- `+ NaOPh → [NP(OPh)₂]ₙ`

---

#### 🔵 **روش‌های دیگر** (مختصر بدان):

1. **پلیمریزاسیون فسفازن‌های آلی‌دار**
   - ابتدا گروه‌های آلی را به حلقه بزن
   - سپس حلقه را پلیمریزه کن

2. **پلیمریزاسیون تراکمی**
   - `PCl₅ + NH₃ → زنجیرهای کوتاه`
   - یا استفاده از `Me₃SiN=PCl₃` (Living polymerization)

3. **تجزیه آزیدها**
   - `R₂PN₃ --[گرما]--> پلیمر + N₂`

---

### 3️⃣ **مکانیسم Ring-Opening (مهم برای درک)**

**مشاهدات تجربی**:
1. نیاز به اتم‌های هالوژن روی حلقه
2. یون‌ها در فرآیند شرکت دارند
3. ناخالصی‌ها (H₂O، BCl₃) کاتالیزور هستند
4. `(NPF₂)₃` دمای بالاتری نیاز دارد (۳۵۰°C) نسبت به `(NPCl₂)₃` (۲۱۰°C)

**مکانیسم پیشنهادی**: کاتیونی
```
P-Cl → P⁺ + Cl⁻
P⁺ حلقه را باز می‌کند و زنجیر را ادامه می‌دهد
```

**عامل مهم دیگر**: **مانع استریک (فضایی)**
- گروه‌های بزرگ (مثل Ph) در پلیمر به هم فشار می‌آورند
- پلیمر ناپایدار می‌شود
- برای همین `(NPPh₂)₃` پلیمریزه نمی‌شود!

---

### 4️⃣ **ساختار مولکولی**

#### **پیوندها در زنجیره**:
```
—N═P—N═P—
```

**نکته کلیدی**: این پیوندها مثل پیوند دوگانه آلی نیستند!
- همه پیوندها تقریباً هم طول هستند
- مدل: `dπ-pπ` bonding (نه `pπ-pπ`)
- الکترون‌های N در orbital 2pz
- الکترون‌های P در orbital 3d
- **نتیجه**: مانع چرخشی خیلی کم (≈ ۰.۱ kcal/mol)

#### **کانفورماسیون زنجیر**:
**معمولاً**: cis-trans-planar
```
    R R   R R
    | |   | |
—N-P-N-P-N-P—
```
- گروه‌های R از هم دورترین فاصله را دارند
- کمترین تنش → پایدارترین حالت
- با XRD تایید شده (فاصله تکرار ≈ ۴.۹ Å)

---

### 5️⃣ **رابطه ساختار-خواص (خیلی مهم!)**

#### 🌡️ **دمای گذار شیشه‌ای (Tg)**

**قانون کلی**:
- گروه‌های کوچک و انعطاف‌پذیر → Tg پایین
- گروه‌های بزرگ و سخت → Tg بالا

**مثال‌ها** (حفظ کن!):
| پلیمر | Tg (°C) | خاصیت |
|--------|---------|-------|
| `(NPF₂)ₙ` | -96 | الاستومر |
| `(NPCl₂)ₙ` | -66 | الاستومر |
| `[NP(OCH₂CF₃)₂]ₙ` | -66 | الاستومر/فیبر |
| `[NP(OPh)₂]ₙ` | -8 | ترموپلاستیک |
| `[NP(NHPh)₂]ₙ` | +91 | شیشه |
| `[NP(NH-adamantyl)(OCH₂CF₃)]ₙ` | +180 | شیشه |

#### 💧 **آبگریزی vs آبدوستی**

**آبگریز** (hydrophobic):
- `—OCH₂CF₃` (زاویه تماس ≈ ۱۰۰-۱۰۷°)
- `—OPh`
- `—Si(CH₃)₃`

**آبدوست** (hydrophilic):
- `—NHCH₃` (زاویه تماس < ۳۰°)
- `—OCH₂CH₂OCH₃`
- `—OC₆H₄COOH`

**حل در آب**:
- `[NP(NHCH₃)₂]ₙ`
- `[NP(OCH₂CH₂OCH₂CH₂OCH₃)₂]ₙ` (MEEP)

#### 🧪 **پایداری در برابر آب**

**پایدار**:
- بیشتر پلی‌فسفازن‌های آلی

**قابل تجزیه** (bioerodible):
- `[NP(NHCH₂COOR)₂]ₙ` (استرهای آمینواسید)
- `[NP(imidazolyl)₂]ₙ`
- `[NP(glyceryl)₂]ₙ`
- `[NP(glucosyl)₂]ₙ`

---

### 6️⃣ **کاربردها (حتماً بدان!)**

#### 🔧 **الاستومرهای پیشرفته**
- `[NP(OCH₂CF₃)(OCH₂(CF₂)ₓCF₂H)]ₙ`
- مقاوم در برابر روغن و سوخت
- انعطاف در دمای پایین (-60°C)
- ضد حریق
- **کاربرد**: صنایع هوافضا، خودرو

#### 🧬 **کاربردهای پزشکی**

**۱. مواد بی‌اثر (Bioinert)**:
- `[NP(OCH₂CF₳)₂]ₙ`
- `[NP(OPh)₂]ₙ`
- سطح آبگریز → سازگاری با خون
- **کاربرد**: دریچه قلب، رگ مصنوعی

**۲. هیدروژل‌ها**:
- MEEP cross-linked
- جذب آب > ۹۵٪
- **کاربرد**: عدسی چشم، رهش دارو

**۳. پلیمرهای زیست‌تخریب‌پذیر**:
- `[NP(NHCH₂COOEt)₂]ₙ` (گلیسین استر)
- تجزیه → آمینواسید + فسفات (غیرسمی!)
- **کاربرد**: 
  - بخیه جراحی
  - مهندسی بافت
  - رهش کنترل‌شده دارو

**۴. میکروکپسول‌ها**:
- PCPP = `[NP(OC₆H₄COOH)₂]ₙ`
- Cross-link با Ca²⁺
- **کاربرد**: واکسن خوراکی آنفولانزا

#### ⚡ **رساناهای یونی**
- MEEP + LiSO₃CF₃
- رسانایی > ۱۰۰۰ برابر PEO
- **کاربرد**: باتری لیتیومی

#### 🔬 **غشاها**
- `[NP(OPh)₂]ₙ` → جداسازی SO₂
- `[NP(OCH₂CF₃)₂]ₙ` → جداسازی CO₂
- هیدروژل‌ها → غشاهای پاسخ‌دهنده (pH، دما)

#### 💡 **مواد نوری**
- کریستال مایع
- غیرخطی نوری (NLO)
- ضریب شکست بالا (تا ۱.۷۵)

## 📝 نکات مهم برای امتحان

### ✅ **چک‌لیست مرور**:

1. ☑️ **تاریخچه**: استوکس، آلکوک، "لاستیک معدنی"
2. ☑️ **ساختار**: `—N═P—` با R روی P
3. ☑️ **سنتز**: Ring-opening + جایگزینی ماکرومولکولی
4. ☑️ **مکانیسم**: کاتیونی، نقش هالوژن، مانع استریک
5. ☑️ **پیوندها**: dπ-pπ (نه pπ-pπ)
6. ☑️ **کانفورماسیون**: cis-trans-planar
7. ☑️ **Tg**: وابسته به اندازه R
8. ☑️ **آبگریزی/آبدوستی**: وابسته به نوع R
9. ☑️ **کاربردها**: الاستومر، پزشکی، باتری، غشا، نوری

### 🎯 **سوالات احتمالی**:

**۱. چرا `(NPCl₂)ₙ` باید قبل از cross-linking سنتز شود؟**
- پاسخ: برای حل شدن در حلال و امکان واکنش جایگزینی

**۲. تفاوت پلی‌فسفازن با پلیمر آلی چیست؟**
- زنجیره معدنی P-N
- پیوند دوگانه متفاوت (dπ-pπ)
- انعطاف بیشتر
- تنوع خواص بیشتر

**۳. چگونه Tg را کنترل می‌کنیم؟**
- گروه‌های کوچک → Tg پایین
- گروه‌های بزرگ → Tg بالا

**۴. چرا برای کاربردهای پزشکی مناسب است؟**
- تنوع خواص
- سازگاری زیستی
- قابلیت زیست‌تخریب
- محصولات تجزیه غیرسمی

**۵. MEEP چیست و چرا مهم است؟**
- `[NP(OCH₂CH₂OCH₂CH₂OCH₃)₂]ₙ`
- حل در آب
- هیدروژل
- رسانای یونی عالی
```

۱.۱ تعریف و ساختار کلی

پلی‌فسفازن‌ها بزرگترین و متنوع‌ترین خانواده از ماکرومولکول‌های معدنی هستند. تا به امروز بیش از ۷۰۰ نوع مختلف از این پلیمرها سنتز شده که دامنه وسیعی از خواص فیزیکی و شیمیایی را در بر می‌گیرند - دامنه‌ای که تا پیش از این فقط برای پلیمرهای آلی سنتتیک شناخته شده بود.

ساختار مولکولی عمومی:

        R         R         R         R
        |         |         |         |
    —N═P—N═P—N═P—N═P—N═P—
        |         |         |         |
        R         R         R         R
```

**اجزای ساختاری:**
- **زنجیره اصلی (Backbone)**: اتم‌های فسفر و نیتروژن به صورت متناوب
- **گروه‌های جانبی (R)**: دو گروه جانبی به هر اتم فسفر متصل است
- **نوع گروه‌های جانبی**: می‌توانند واحدهای آلی، فلز-آلی، یا معدنی باشند

**ابعاد مولکولی:**
- **تعداد واحدهای تکراری**: معمولاً از ۱۰۰ تا ۱۵,۰۰۰ یا بیشتر
- **وزن مولکولی**: بسته به گروه‌های آلی جانبی، بالاترین وزن‌های مولکولی در محدوده ۲ میلیون تا ۱۰ میلیون قرار دارند

### ۱.۲ ویژگی منحصر به فرد پیوندها

**نکته بسیار مهم برای امتحان:**

ساختار پیوندی در زنجیره اصلی به صورت رسمی به عنوان یک سری پیوند یک‌گانه و دوگانه متناوب نشان داده می‌شود:
```
—N═P—N═P—N═P—
```

**اما این نمایش گمراه‌کننده است!** 

اندازه‌گیری‌های ساختاری نشان می‌دهند که:
1. **همه پیوندها در طول زنجیر تقریباً برابر یا نزدیک به هم هستند**
2. **کونژوگاسیون گسترده‌ای مانند مواد پلی‌غیراشباع آلی وجود ندارد**
3. این موضوع بعداً به تفصیل بحث خواهد شد

## 📖 بخش دوم: تاریخچه تفصیلی (برای امتحان بسیار مهم!)

### ۲.۱ دوره اولیه: کشف و توصیف (۱۸۳۴-۱۸۹۷)

#### **۱۸۳۴: شروع داستان**
- **لیبیگ (Liebig)** و **روز (Rose)** اولین بار گزارش دادند که پنتاکلرید فسفر با آمونیاک واکنش می‌دهد
- محصول: یک ترکیب سفید رنگ و کریستالی که در حلال‌های آلی محلول بود

#### **تحقیقات بعدی (۱۸۴۶-۱۸۷۰)**
پژوهشگران مختلف روی این ترکیب کار کردند:
- **Gerhardt** (۱۸۴۶)
- **Laurent** (۱۸۵۰)
- **Gladstone و Holmes** (۱۸۶۴)
- **Wichelhaus** (۱۸۷۰)

این محققان فرمول ترکیب را به عنوان **N₃P₃Cl₆** تعیین کردند و همچنین یک تترامر مشابه با فرمول **N₄P₄Cl₈** را جداسازی و مطالعه کردند.

#### **۱۸۹۷: کار پیشگامانه استوکس (H. N. Stokes)**

**استوکس** محقق آمریکایی، پیشنهاد داد که:
- N₃P₃Cl₆ دارای ساختار حلقوی است:
```
        Cl    Cl
         |    |
    Cl—P——N——P—Cl
        |    |
        N    N
        |    |
        P——  
        |
       Cl Cl
```

- N₄P₄Cl₈ نیز ساختار حلقوی مشابهی دارد (۸ عضوی)
- او همچنین حلقه‌های بزرگتر مانند (NPCl₂)₅₋₇ را جداسازی کرد

**واکنش کلی سنتز:**
```
nPCl₅ + nNH₄Cl → (NPCl₂)ₙ + 4nHCl
```

### ۲.۲ کشف "لاستیک معدنی" (Inorganic Rubber)

**استوکس اولین کسی بود که شکل پلیمری این ترکیبات را تهیه کرد:**

**روش:**
- هر کدام از ترکیبات حلقوی کلروفسفازن کوچک را گرم کرد
- آنها به یک ماده لاستیکی تبدیل شدند
- این ماده بعداً "لاستیک معدنی" نامیده شد

**خواص لاستیک معدنی:**
1. در حلال‌های آلی مانند بنزن **متورم می‌شد اما حل نمی‌شد**
2. با گرم کردن در فشار کم، تجزیه شده و دوباره به ترکیبات حلقوی تبدیل می‌شد
3. در اتمسفر مرطوب به فسفات آمونیوم و اسید کلریدریک هیدرولیز می‌شد

**چرا حل نمی‌شد؟** (نکته کلیدی)
- ماده cross-linked (شبکه‌ای) بود
- زنجیرهای پلیمری به یکدیگر متصل شده بودند

**اهمیت تاریخی:**
از مقالات استوکس کاملاً مشخص است که او تشخیص داده بود لاستیک معدنی یک **پلیمر** است - احتمالاً ادامه سری ترکیبات حلقوی که جداسازی کرده بود. اما:
- احتمالاً نمی‌دانست که این یک **پلیمر سنگین** با هزاران واحد تکراری است
- چون ایده وجود پلیمرهای سنگین تا ۴۰ سال بعد (کار استاودینگر) پذیرفته نشد

### ۲.۳ دوره رکود (۱۸۹۷-۱۹۳۰)

**۴۰ سال بعد**: لاستیک معدنی فقط به صورت پراکنده به عنوان یک کنجکاوی آزمایشگاهی ذکر شد

**چرا؟**
- پذیرش کند ایده بحث‌برانگیز **استاودینگر** (۱۹۲۰) که ماکرومولکول‌های زنجیر بلند می‌توانند وجود داشته باشند
- عدم حلالیت لاستیک معدنی
- ناپایداری هیدرولیتیکی آن

### ۲.۴ احیای علاقه (۱۹۲۰-۱۹۳۶)

**۱۹۲۸: Meyer و Mark**
- نشان دادند که لاستیک طبیعی یک ماکرومولکول خطی است

**۱۹۳۶: Meyer، Lotmar و Pankow**
- آزمایش‌های پراش اشعه X
- قویاً نشان دادند که لاستیک معدنی حاوی زنجیرهای پلیمری خطی است
- واحد تکراری از نوع نشان داده شده در ساختار عمومی

**اما هنوز مشکل بود:**
- عدم حلالیت در همه حلال‌ها
- ناپایداری هیدرولیتیکی در اتمسفر
- علاقه جدی به این ماده کاهش یافت

### ۲.۵ انقلاب مدرن (۱۹۶۵-۱۹۶۶): کار آلکوک

**Allcock، Kugel و Valan** سه مقاله منتشر کردند که نقطه شروع رویکرد مدرن به پلی‌فسفازن‌ها شد.

#### **تفکر اصلی:**

**۱. درک ناپایداری:**
- ناپایداری هیدرولیتیکی لاستیک معدنی = نشانه واکنش‌پذیری بالای پیوندهای P-Cl
- **ایده**: شاید بتوان از این واکنش‌پذیری استفاده کرد
- اتم‌های کلر را با گروه‌های آلی جایگزین کنیم → پلیمرهای پایدار هیدرولیتیکی

**تلاش‌های اولیه:**
- سعی شد کلرها در لاستیک معدنی با گروه‌های آلکوکسی جایگزین شوند
- نتیجه دلگرم‌کننده بود
- **اما**: جایگزینی هرگز کامل نمی‌شد (به خاطر عدم حلالیت)

**۲. حل مشکل حلالیت:**

**تحلیل:**
- عدم حلالیت + متورم شدن = ویژگی پلیمرهای cross-linked
- ساختار لاستیک معدنی احتمالاً این است:
```
        Cl    Cl         Cl    Cl
        |     |          |     |
    —N═P—N═P—N═P—N═P—N═P—N═P—
        |     |    \     |     |
        Cl    Cl    \    Cl    Cl
                     \
                      \—P—N—
                         |
```

**راه حل:**
باید روشی پیدا کرد برای تهیه پلیمر در حالت **بدون cross-link**

**کشف کلیدی:**
- مطالعه دقیق فرآیند تبدیل هگزاکلروسیکلوتری‌فسفازن به پلی(دی‌کلروفسفازن)
- کنترل دقیق: زمان، دما، خلوص تریمر حلقوی
- **خیلی مهم**: متوقف کردن واکنش **قبل از رسیدن به تبدیل ~۷۰٪** به پلیمر

**نتیجه:**
پلیمر خطی (بدون cross-link) که **کاملاً در حلال‌های آلی حل می‌شد**:
- بنزن
- تولوئن  
- تتراهیدروفوران

**اما**: گرم کردن بیشتر این پلیمر → cross-linking → لاستیک معدنی نامحلول

**واکنش کلی:**
```
(NPCl₂)₃ --[250°C، کنترل شده]--> (NPCl₂)ₙ --[حلال]--> محلول
  حلقه                               خطی                پلیمر حل‌شده

(NPCl₂)ₙ --[گرمای بیشتر]--> پلیمر cross-linked
  خطی                          نامحلول
```

**۳. واکنش‌های جایگزینی ماکرومولکولی:**

پلی(دی‌کلروفسفازن) محلول شده به عنوان یک **واکنش‌دهنده ماکرومولکولی قابل توجه** عمل کرد:

**روش:**
```
(NPCl₂)ₙ + نوکلئوفیل آلی → پلیمر جایگزین شده
```

**نوکلئوفیل‌ها:**
- نمک‌های سدیمی الکل‌ها (NaOR)
- نمک‌های سدیمی فنل‌ها (NaOAr)
- آمین‌های اولیه یا ثانویه (RNH₂، R₂NH)

**نکته شگفت‌انگیز:**
به طور متوسط **۳۰,۰۰۰ اتم کلر در هر مولکول** جایگزین می‌شدند!

**مثال واکنش:**
```
(NPCl₂)ₙ + NaOCH₂CF₃ --[THF]--> [NP(OCH₂CF₃)₂]ₙ + NaCl↓
```

**۴. خواص پلیمرهای مشتق شده:**

پلیمرهای مشتق شده:
- **پایدار هیدرولیتیکی** بودند
- دامنه وسیعی از خواص غیرمعمول، جالب و کاربردی داشتند
- **Tg** و **Tm** با گروه‌های جانبی مختلف، تغییر می‌کرد
- حلالیت پلیمر متفاوت بود
- خواص سطحی متنوع
- زنجیره پلیمری در برابر گرما و تابش انرژی بالا پایدار بود

**نتیجه‌گیری اولیه:**
حتی در مراحل اولیه توسعه این حوزه، مشخص بود که می‌توان **دامنه وسیعی از کاربردها** را از این پلیمرها توسعه داد.

**پیشرفت‌های بعدی:**
طی ۳۵ سال بعد، بیش از **۷۰۰ پلی‌فسفازن مختلف** با گروه‌های جانبی و معماری‌های اسکلتی مختلف سنتز شدند.

### ۲.۶ اهمیت تاریخی تهیه پلی(دی‌کلروفسفازن) بدون cross-link

**این مهم‌ترین عامل در توسعه اولیه شیمی پلی‌فسفازن بود.**

**مقایسه با پلیمرهای آلی:**

**پلیمرهای آلی معمولی:**
- تنوع ساختاری با پلیمریزاسیون مونومرهای مختلف حاصل می‌شود
- برای هر پلیمر آلی جدید، باید روشی برای پلیمریزاسیون هر مونومر جدید پیدا کرد
- این شاید **بزرگترین چالش** در شیمی پلیمر آلی معمولی است

**پلی‌فسفازن‌ها:**
- تغییرات قابل توجه ساختاری و خواصی با تغییر در شیمی جایگزینی **فقط یک پلیمر از پیش ساخته شده** حاصل می‌شود
- دسترسی آسان به تعداد زیادی ماکرومولکول مختلف

**چرا این روش در پلیمرهای آلی کار نمی‌کند؟**
خیلی کم پلیمرهای آلی معمولی واکنش‌پذیری لازم را برای این رویکرد دارند.

**شناخت کلیدی:**
پیوندهای P-Cl **بسیار واکنش‌پذیرتر** به حمله نوکلئوفیلی هستند نسبت به:
- پیوندهای C-H در پلیمرهای آلی
- پیوندهای C-Cl در پلیمرهای آلی

**نتیجه:**
تهیه پلی(دی‌کلروفسفازن) بدون cross-link + نشان دادن واکنش‌پذیری آن = **آزمایش‌های کلیدی** که از آنها این حوزه منشعب شد

## 🔬 بخش سوم: روش‌های سنتز جایگزین برای پلیمرهای خطی

### ۳.۱ خلاصه روش‌های موجود

پلی(ارگانوفسفازن)های خطی اکنون با **شش روش مختلف** سنتز می‌شوند:

**الف) پلیمریزاسیون باز شدن حلقه + جایگزینی ماکرومولکولی**
- پلیمریزاسیون (NPCl₂)₃
- سپس جایگزینی کلرها

**ب) پلیمریزاسیون تریمرهای حلقوی جایگزین‌شده آلی**
- ابتدا گروه‌های آلی به حلقه وصل می‌شوند
- سپس حلقه پلیمریزه می‌شود

**ج) پلیمریزاسیون تراکمی**
- تولید (NPCl₂)ₙ
- سپس جایگزینی ماکرومولکولی
- شامل سنتز کوپلیمرهای هیبریدی block و graft

**د) پلیمریزاسیون تراکمی فسفورانیمین‌های N-سیلیل‌دار**
- مثل Me₃SiN=PR₂Cl
- مستقیماً به پلیمرهای جایگزین‌شده آلی

**ه) تجزیه کنترل‌شده آزیدهای ارگانوفسفر**
- مثل R₂PN₃

**و) واکنش‌های ثانویه**
- روی گروه‌های جانبی آلی که از قبل در زنجیر پلی‌فسفازن وجود دارند

### ۳.۲ روش باز شدن حلقه/جایگزینی ماکرومولکولی (مهم‌ترین روش)

#### **اهمیت:**
این اولین روش سنتزی است و **گسترده‌ترین روش** برای تهیه پلی(ارگانوفسفازن)ها است.

#### **مرحله ۱: پلیمریزاسیون باز شدن حلقه**

**تهیه هگزاکلروسیکلوتری‌فسفازن:**
```
3PCl₅ + 3NH₄Cl --[حلال آلی]--> (NPCl₂)₃ + 12HCl
                   کلروبنزن
```
- این ترکیب در مقیاس صنعتی تهیه می‌شود
- نیاز به خالص‌سازی دقیق و حفاظت از رطوبت

**پلیمریزاسیون:**
```
(NPCl₂)₃ --[210-250°C، مذاب]--> (NPCl₂)ₙ
```

**شرایط:**
- گرم کردن در حالت مذاب
- دمای ۲۱۰ تا ۲۵۰ درجه سانتیگراد
- کنترل دقیق برای جلوگیری از cross-linking

**قابلیت کاربرد روی ترکیبات دیگر:**
این فرآیند با تغییرات جزئی قابل اعمال است روی:

1. **فسفازو-فسفازن‌های حلقوی:**
```
    Cl₂P—NPCl₃
     |      |
     N      N
     |      |
    Cl₂P—N—P—Cl
           |
          NPCl₃
```

2. **فلوروفسفازن‌های حلقوی:**
```
(NPF₂)₃ → (NPF₂)ₙ
(NPF₂)₄ → (NPF₂)ₙ
```

3. **کربو-فسفازن‌ها:**
```
    Cl₂P—N=C
     |      |
     N      N
     |      |
    Cl₂P—N—P—Cl
```

4. **تیو-فسفازن‌ها:**
```
(NSCl₂)₃ → پلیمر
```

5. **تیونیل-فسفازن‌ها:**
```
[NS(O)Cl₂]₃ → پلیمر
```

#### **مرحله ۲: جایگزینی ماکرومولکولی**

**واکنش عمومی:**
```
(NPCl₂)ₙ + نوکلئوفیل --[حلال]--> پلیمر جایگزین شده + NaCl↓
```

**محرک واکنش:**
- رسوب NaCl از محلول
- این فرآیند واکنش را به سمت کامل شدن هدایت می‌کند

**مثال‌های مهم واکنش:**

**۱. با تری‌فلوئورواتوکسید سدیم:**
```
(NPCl₂)ₙ + 2n NaOCH₂CF₃ → [NP(OCH₂CF₃)₂]ₙ + 2n NaCl↓
```
- **پلیمر ۳.۲۹**: اولین پلی(ارگانوفسفازن) پایدار که سنتز شد
- هنوز یکی از جالب‌ترین‌ها است

**۲. با اتوکسید:**
```
(NPCl₂)ₙ + 2n NaOC₂H₅ → [NP(OC₂H₅)₂]ₙ + 2n NaCl
```

**۳. با فنوکسید:**
```
(NPCl₂)ₙ + 2n NaOPh → [NP(OPh)₂]ₙ + 2n NaCl
```

**۴. با متیل‌آمین:**
```
(NPCl₂)ₙ + 4n CH₃NH₂ → [NP(NHCH₃)₂]ₙ + 2n CH₃NH₃Cl
```

**۵. با دی‌اتیل‌آمین:**
```
(NPCl₂)ₙ + 4n (C₂H₅)₂NH → [NP(N(C₂H₅)₂)₂]ₙ + 2n (C₂H₅)₂NH₂Cl
```

**۶. با آنیلین:**
```
(NPCl₂)ₙ + 4n C₆H₅NH₂ → [NP(NHC₆H₅)₂]ₙ + 2n C₆H₅NH₃Cl
```

#### **محدودیت‌های روش:**

**مانع استریک (فضایی):**
- گروه‌های جانبی حجیم می‌توانند جایگزینی کلرهای نزدیک را کند کنند
- مثال: واحدهای آریلوکسی چند حلقه‌ای یا ortho-substituted
- ممکن است نیاز به شرایط سخت باشد (دمای بالا، راکتور بسته)

**آمینولیز:**
```
(NPCl₂)ₙ + RNH₂ → [NP(NHR)₂]ₙ
```
- مانع استریک می‌تواند جایگزینی آخرین ۲۵٪ کلرها را کند کند
- نیاز به شرایط سخت در انتهای واکنش

#### **پلیمرهای با جانشین‌های مختلط:**

**اصل مهم:**
دو یا چند گروه آلی مختلف را می‌توان **همزمان یا متوالی** وارد کرد تا پلیمرهای با جانشین‌های مختلط تولید شود.

**روش ۱: جایگزینی متوالی با استفاده از مانع استریک**

**مثال با دی‌اتیل‌آمین:**
```
(NPCl₂)ₙ + n (C₂H₅)₂NH → [NP(Cl)[N(C₂H₅)₂]]ₙ
                          (فقط یک Cl جایگزین می‌شود)

[NP(Cl)[N(C₂H₅)₂]]ₙ + n CH₃NH₂ → [NP(NHCH₃)[N(C₂H₅)₂]]ₙ
```

**توضیح:**
- دی‌اتیل‌آمین حتی در مقدار زیاد فقط **یک کلر** روی هر فسفر را جایگزین می‌کند (به خاطر حجم زیاد)
- کلرهای باقیمانده با نوکلئوفیل کمتر حجیم (متیل‌آمین) جایگزین می‌شوند

**روش ۲: جایگزینی متوالی با واکنش‌پذیری متفاوت**

**مثال با پلیمرهای فسفازو:**
```
    Cl₂P—NPCl₃        R₂P—NPCl₃
     |      |          |      |
—N—P—N—P—N—  + RMgX → —N—P—N—P—N—
     |      |          |      |
    Cl      Cl        Cl      Cl

سپس + R'ONa → پلیمر با R و OR' مختلط

توضیح: کلرهای گروه جانبی واکنش‌پذیرتر از کلرهای روی زنجیر اصلی هستند → ابتدا جایگزین می‌شوند

روش ۳: استفاده از واکنش‌دهنده‌های ارگانوفلزی

واکنش (NPCl₂)ₙ با RMgX یا RLi پیچیده‌تر است:

دو مسیر رقیب:

(NPCl₂)ₙ + RLi → جایگزینی Cl با R

شکستن پیوندهای P-N در اسکلت


**توضیح مکانیسم شکستن زنجیر:**

**عامل کلیدی**: هماهنگی (coordination) مولکول‌های ارگانوفلزی با اتم‌های نیتروژن اسکلتی
```
        R         R
        |         |
    —N═P—N═P—N═P—
        |    ↑    |
        R    |    R
             |
           RLi (هماهنگی)
             ↓
        شکستن پیوند P-N
```

**عوامل موثر بر شکستن زنجیر:**

**۱. الکترون‌های جفت‌نشده نیتروژن:**
- هر ساختاری که coordination به N را تقویت کند → شکستن اسکلت را تشویق می‌کند
- این به چگالی الکترون‌های جفت‌نشده روی N بستگی دارد

**۲. نقش کلر به عنوان گروه الکترون‌کش:**
- کلرها واحدهای الکترون‌کش عالی هستند
- چگالی الکترون‌های جفت‌نشده روی نیتروژن اسکلتی را کاهش می‌دهند
- اسکلت را در برابر شکستن محافظت می‌کنند

**۳. خطر پس از جایگزینی:**
اگر کلرها با گروه‌های آلکیل یا آریل جایگزین شوند:
- اثر محافظتی از بین می‌رود
- احتمال شکستن پیوند P-N افزایش می‌یابد

**نتیجه:**
هر چه واکنش جایگزینی ارگانوفلزی پیش می‌رود، احتمال بیشتری وجود دارد که شکستن پیوند P-N رخ دهد به جای جایگزینی هالوژن.

#### **راه حل: استفاده از فلوروفسفازن‌ها**

**استدلال:**
فلوئور واحد **الکترون‌کش‌تری** از کلر است → باید مقاومت بیشتری در برابر شکستن اسکلتی داشته باشد.

**شواهد تجربی** (شکل ۳.۴):

مقایسه تغییرات وزن مولکولی متوسط برای:
- `(NPCl₂)ₙ + PhLi`
- `(NPF₂)ₙ + PhLi`

**نتایج:**
- **با (NPCl₂)ₙ**: شکستن اسکلتی تقریباً از همان ابتدای واکنش قابل تشخیص است
- **با (NPF₂)ₙ**: شکستن زنجیر قابل توجه نیست تا زمانی که تقریباً ۷۵٪ اتم‌های فلوئور با گروه‌های آلی جایگزین شوند

**تهیه (NPF₂)ₙ:**
```
(NPF₂)₃ --[پلیمریزاسیون]--> (NPF₂)ₙ
   ↑
   |
(NPCl₂)₃ + NaF
```

**محدودیت (NPF₂)ₙ:**
- **عدم حلالیت** در همه حلال‌ها به جز محیط‌های فلوروکربنی تخصصی
- این مشکل کاربرد آن را محدود می‌کند

**راه حل عملی:**
استفاده از پلیمر فلوئوردار با یک گروه جانبی فنیل برای هر سه واحد تکراری:
```
[NP(F)₂]₃-[NP(Ph)(F)]₁ → واحد تکراری
```

**مزایا:**
- این ماکرومولکول در چندین حلال آلی معمولی محلول است
- بستر عالی برای واکنش‌های جایگزینی فلوئور

**تهیه:**
```
(NPF₂)ₙ + مقدار کم PhLi → [NP(F)₂]₃[NP(Ph)(F)]ₙ

[NP(F)₂]₃[NP(Ph)(F)]ₙ + RMgX → جایگزینی F بدون شکستن زنجیر
```

#### **تکنیک پیشرفته: محافظت از اسکلت**

**اصل:**
اگر **بیشتر** اتم‌های کلر یا فلوئور با گروه‌های آلی **الکترون‌کش** (مثل تری‌فلوئورواتوکسی) جایگزین شوند، کلر یا فلوئورهای باقیمانده را می‌توان با واکنش‌دهنده‌های ارگانوفلزی **بدون شکستن قابل توجه اسکلت** جایگزین کرد.

**مثال:**
```
(NPCl₂)ₙ + NaOCH₂CF₃ (زیاد) → [NP(OCH₂CF₃)(Cl)]ₙ
                                  (بیشتر Cl‌ها جایگزین شده)

[NP(OCH₂CF₃)(Cl)]ₙ + RLi → [NP(OCH₂CF₃)(R)]ₙ
                            (بدون شکستن زنجیر)
```

**کاربرد:**
این تکنیک مفید برای تهیه پلیمرهای با جانشین‌های مختلط است که شامل:
- گروه‌های آلکیل
- گروه‌های آریل
- گروه‌های کاربورانیل
- گروه‌های ارگانوفلزی فلزات واسطه

#### **جمع‌بندی مزایا و محدودیت‌ها**

**علی‌رغم پیچیدگی‌ها:**
روش جایگزینی ماکرومولکولی کلی دسترسی به **دامنه عظیمی** از ساختارهای پلیمری مختلف را فراهم می‌کند.

**ویژگی مهم:**
طول زنجیر پلیمر معمولاً تحت تأثیر جایگزینی هالوژن با گروه‌های جانبی آلی **قرار نمی‌گیرد**.

**اهمیت برای تحقیق:**
امکان تغییر ساختار گروه جانبی در پلی‌فسفازن‌ها **بدون تغییر هیچ ویژگی ساختاری دیگر** → اهمیت حیاتی برای ارزیابی روابط ساختار-خاصیت

### ۳.۳ پلیمریزاسیون فسفازن‌های حلقوی جایگزین‌شده آلی یا فلزی-آلی

#### **انگیزه:**

روش جایگزینی ماکرومولکولی قدرتمند است، **اما** محدودیت دارد برای پلیمرهای هدف که باید حاوی گروه‌های جانبی آلی متصل به اسکلت از طریق **پیوندهای کربن-فسفر** باشند.

**مشکل:**
جایگزینی‌های ارگانوفلزی ماکرومولکولی اجازه می‌دهند گروه‌های آلکیل، آریل یا ارگانوفلزی را وارد کنند، اما **اغلب به بهای شکستن اسکلتی یا جایگزینی ناقص هالوژن**.

**راه حل:**
روش‌های این بخش و بخش بعدی رویکردهای جایگزینی را ارائه می‌دهند که از تعامل واکنش‌دهنده‌های ارگانوفلزی با پلی‌فسفازن‌های سنگین اجتناب می‌کنند.

#### **اصل روش:**

**انجام واکنش‌های جایگزینی ارگانوفلزی در سطح تریمر حلقوی کوچک آسان‌تر است** نسبت به پلیمرهای سنگین مشابه.

**دلیل:**
پیامدهای واکنش‌های شکستن اسکلتی در شیمی مولکول کوچک کمتر شدید است نسبت به پلیمرهای سنگین.

**مثال محاسباتی مهم:**

فرض کنید تعامل یک واکنش‌دهنده ارگانوفلزی (مثل RLi) با یک کلروفسفازن به طور متوسط منجر به:
- **یک واکنش شکستن اسکلتی** برای هر **دوازده واکنش جایگزینی هالوژن** می‌شود

**در سطح پلیمر سنگین:**
```
نسبت = 1 شکستن : 12 جایگزینی
     = 1 شکستن : 6 واحد تکراری
     
نتیجه: مولکول‌ها فقط 6 واحد تکراری طول دارند
       → هیچ خاصیت پلیمری ندارند
       → بازده پلیمر = صفر
```

**در سطح تریمر حلقوی:**
```
نسبت = 1 شکستن : 12 جایگزینی
     = 1 حلقه شکسته : 2 حلقه سالم
     
نتیجه: بازده تریمر جایگزین‌شده حلقوی = 50%
       → قابل قبول برای سنتز
```

**نتیجه‌گیری:**
تهیه تریمرهای حلقوی ارگانو-جایگزین‌شده با شیمی جایگزینی ارگانوفلزی **بسیار آسان‌تر** است از تهیه پلیمرهای سنگین مشابه.

#### **استراتژی دو مرحله‌ای:**

**مرحله ۱: جایگزینی در سطح حلقه**
```
(NPCl₂)₃ + RLi یا RMgX → (NPR₂)ₓ(NPCl₂)₃₋ₓ
```
- معرفی گروه‌های آلی یا ارگانوفلزی در سطح تریمر حلقوی
- شکستن حلقه مشکل کمتری ایجاد می‌کند

**مرحله ۲: پلیمریزاسیون باز شدن حلقه**
```
(NPR₂)ₓ(NPCl₂)₃₋ₓ --[گرما]--> پلیمر
```
- تریمر جایگزین‌شده را پلیمریزه کن
- سپس (در صورت لزوم) جایگزینی ماکرومولکولی روی هالوژن‌های باقیمانده

**واکنش کلی:**
```
مرحله 1:
(NPCl₂)₃ + گروه‌های آلی → حلقه جایگزین‌شده

مرحله 2:
حلقه جایگزین‌شده --[250°C]--> پلیمر خطی
```

#### **مزایا و محدودیت‌ها:**

**مزایا:**
1. دسترسی به پلیمرهای با پیوند C-P
2. کنترل بهتر روی ترکیب
3. از شکستن زنجیر پلیمری جلوگیری می‌شود

**محدودیت‌ها:**

**قانون کلی:**
تریمرهای حلقوی که دارای **یک یا دو** گروه جانبی آلی/ارگانوفلزی و **پنج یا چهار** اتم کلر یا فلوئور هستند، معمولاً **تقریباً به راحتی** (NPCl₂)₃ یا (NPF₂)₃ پلیمریزه می‌شوند.

**اما:**
تمایل به پلیمریزاسیون کاهش می‌یابد هر چه **بیشتر** اتم‌های هالوژن در تریمر با گروه‌های آلی جایگزین شوند.

**مثال کلاسیک: (NPMe₂)₃**
```
    Me  Me
     |  |
  Me-P—N—P-Me
     |     |
     N     N
      \   /
       P
      / \
    Me   Me

این حلقه با گرما چه می‌کند؟
```

**نتیجه تجربی:**
```
(NPMe₂)₃ --[گرما]--> (NPMe₂)₄ (تعادل حلقه-حلقه)
                      اما پلیمریزه نمی‌شود!
```

**حلقه‌های دیگر که پلیمریزه نمی‌شوند:**
- `(NPPh₂)₃` (شش گروه فنیل)
- `[NP(OCH₂CF₃)₂]₃` (شش گروه تری‌فلوئورواتوکسی)

**دلایل احتمالی:**
- مانع استریک زیاد
- دمای سقف (ceiling temperature) بالا
- بحث تفصیلی در بخش مکانیسم

#### **استثنا مهم: سیستم‌های ring-strained**

**کشف کلیدی:**
این محدودیت برای حلقه‌های فسفازن که تحت **تنش حلقوی** (ring-strain) توسط یک گروه فروسنیل transannular قرار دارند، صادق نیست.

**مثال: ساختار ۳.۳۷**
```
        R    R
         \  /
    R-P—N—N—P-R
       |  Fe  |
       N      N
        \    /
         P-P
        /    \
       R      R
       
(Fe = هسته فروسن که حلقه را تنش می‌دهد)
```

**واکنش:**
```
ساختار 3.37 --[گرما، +کاتالیزور (NPCl₂)₃]--> پلیمر 3.38
```

**نکته مهم:**
تحت این شرایط، پلیمریزاسیون اتفاق می‌افتد **حتی اگر هیچ اتم هالوژنی** به فسفر متصل نباشد، اگرچه واکنش با حضور مقادیر کاتالیزوری (NPCl₂)₃ تسریع می‌شود.

#### **مثال‌های موفق از پلیمرهای سنتز شده:**

پلیمریزاسیون باز شدن حلقه تریمرهای حلقوی جایگزین‌شده آلی یا ارگانوفلزی برای تهیه طیف وسیعی از پلی‌فسفازن‌های سنگین استفاده شده که در آنها گروه‌های جانبی زیر به زنجیر متصل هستند:

**۱. گروه‌های آلکیل:**
```
(NPR₂)₃ --[گرما]--> (NPR₂)ₙ
R = Me, Et, Pr
```

**۲. گروه‌های آریل:**
```
(NPPh)ₓ(NPCl₂)₃₋ₓ → پلیمر
```

**۳. گروه‌های کاربورانیل:**
```
حلقه با carborane → پلیمر
```

**۴. گروه‌های متالوسنیل:**
```
(NPCl₂)₃ + ferrocene → حلقه → پلیمر فروسنیل‌دار
```

**۵. گروه‌های ارگانوسیلیکون:**
```
(NPCl₂)₃ + R₃Si- → حلقه → پلیمر سیلیکون‌دار
```

**۶. گروه‌های آریلوکسی:**
```
(NPCl₂)₃ + ArONa → حلقه → پلیمر
```

**واکنش‌های نمونه (۱۴-۱۶):**

**واکنش ۱۴:**
```
(NPCl₂)₃ + 2 PhLi → (NPPh₂)(NPCl₂)₂

(NPPh₂)(NPCl₂)₂ --[250°C]--> [NPPh₂][NPCl₂]₂ₙ

[NPPh₂][NPCl₂]₂ₙ + NaOCH₂CF₃ → [NPPh₂][NP(OCH₂CF₃)₂]₂ₙ
```

**واکنش ۱۵:**
```
(NPCl₂)₃ + Me₃SiCH₂MgCl → (NP(CH₂SiMe₃)Cl)(NPCl₂)₂

→ [250°C] → [NP(CH₂SiMe₃)Cl][NPCl₂]₂ₙ

→ [NaOCH₂CF₃] → [NP(CH₂SiMe₃)(OCH₂CF₃)][NP(OCH₂CF₃)₂]₂ₙ
```

**واکنش ۱۶:**
```
(NPCl₂)₃ + گروه carborane → حلقه carborane‌دار

→ [گرما] → پلیمر carborane‌دار
```

**اهمیت:**
نمونه‌هایی از این پلیمرها و خواص و کاربردهای آنها در بخش‌های بعدی بحث خواهد شد.

### ۳.۴ مکانیسم پلیمریزاسیون باز شدن حلقه (خیلی مهم!)

#### **اهمیت:**

از بحث تا اینجا مشخص است که پلیمریزاسیون باز شدن حلقه یک **مرحله حیاتی** در سنتز بیشتر پلی‌فسفازن‌ها است.

**چرا مهم است؟**
1. گسترش دامنه این حوزه
2. بهینه‌سازی شرایط پلیمریزاسیون
3. scale-up این فرآیندها
4. القای پلیمریزاسیون تریمرهای مقاوم مثل (NPMe₂)₃ یا (NPPh₂)₃

#### **مشاهدات تجربی کلیدی:**

تفکر اولیه روی پلیمریزاسیون متمرکز بود روی:
- `(NPCl₂)₃`
- `(NPCl₂)₄`
- آنالوگ فلوئور: `(NPF₂)₃`

**مشاهده ۱: نیاز به هالوژن‌ها**
پلیمریزاسیون به نظر می‌رسد نیاز به حضور **چندین اتم هالوژن** متصل به فسفرهای حلقه دارد.

**شواهد:**
جایگزینی **همه** اتم‌های هالوژن با:
- گروه‌های متیل
- گروه‌های فنیل
- گروه‌های OCH₂CF₃

→ فرآیند پلیمریزاسیون **مسدود می‌شود**
→ (اما تعادل گسترش حلقه مسدود نمی‌شود)

**مشاهده ۲: مشارکت یون‌ها**
یون‌ها به نظر می‌رسد در فرآیند پلیمریزاسیون شرکت می‌کنند.

**شواهد:**
- رسانایی یونی (NPCl₂)₃ مذاب پایین می‌ماند
- تا زمانی که تریمر به محدوده دمایی که پلیمریزاسیون رخ می‌دهد گرم شود
- در این نقطه رسانایی **به طور چشمگیری افزایش می‌یابد**

**مشاهده ۳: اثر کاتالیزوری ناخالصی‌ها**
مقادیر کم ناخالصی، به خصوص:
- آب (H₂O)
- BCl₃
- AlCl₃

اثر **کاتالیزوری** بر پلیمریزاسیون (NPCl₂)₃ دارند.

**مشاهده ۴: تفاوت دمای پلیمریزاسیون**
دمای مورد نیاز برای شروع پلیمریزاسیون:
- `(NPF₂)₃`: **۳۵۰°C**
- `(NPCl₂)₃`: **۲۱۰-۲۵۰°C**

دمای (NPF₂)₃ **بسیار بالاتر** است!

#### **مکانیسم پیشنهادی: کاتیونی**

**بر اساس مشاهدات بالا، مکانیسم زیر پیشنهاد شده:**

**مرحله ۱: یونیزاسیون (Initiation)**
```
    Cl  Cl              Cl  Cl
     |  |                |  |⁺
  Cl-P—N—P-Cl  ⇌  Cl-P—N—P⁺  +  Cl⁻
     |      |          |      |
     N      N          N      N
      \    /            \    /
       P                 P
      / \               / \
    Cl   Cl           Cl   Cl

  (NPCl₂)₃           یون فسفازنیوم حلقوی
```

فرض می‌شود پلیمریزاسیون توسط گونه‌های یونی مانند **۳.۳۹** (یون فسفازنیوم حلقوی یا خطی) که از یونیزاسیون پیوندهای P-Cl تولید می‌شوند، **القا** می‌شود.

**مرحله ۲: شروع و انتشار زنجیر**
```
یون فسفازنیوم⁺ + (NPCl₂)₃ → باز شدن حلقه
                             ↓
                      انتشار زنجیر کاتیونی
```

یون فسفازنیوم به عنوان یک **آغازگر کاتیونی** عمل می‌کند:
1. حمله به اتم نیتروژن اسکلتی یک مولکول (NPCl₂)₃
2. القای باز شدن حلقه
3. انتشار زنجیر با مکانیسم کاتیونی

**واکنش کلی (۱۷):**
```
        Cl  Cl
         |  |
    Cl-P—N—P-Cl  --[گرما]--> Cl-P⁺ + Cl⁻
         |      |               |
         N      N               N
          \    /                 \
           P                      P⁺
          / \                    / \
        Cl   Cl                 

                    ↓ + (NPCl₂)₃
                    
            —N═P—N═P—N═P⁺
             |   |   |
            Cl  Cl  Cl
```

#### **نقش کاتالیزورها:**

**۱. آب (H₂O):**
- ممکن است به عنوان کاتالیزور عمل کند با **کمک به جدایی یون کلرید**
- یا به عنوان گونه همکاتالیزوری عمل کند

**۲. تری‌کلرید بور (BCl₃):**
- به طور معمول به عنوان کاتالیزور در فرآیند تجاری استفاده می‌شود
- انتظار می‌رود جدایی Cl⁻ از فسفر را **ترویج** کند
- به صورت یون BCl₄⁻ پایدار می‌شود:

P-Cl + BCl₃ → P⁺ + BCl₄⁻


**۳. توضیح تفاوت (NPF₂)₃ و (NPCl₂)₃:**
این مکانیسم توضیح می‌دهد چرا (NPF₂)₃ دمای بالاتری برای پلیمریزاسیون نیاز دارد:
- به طور کلی فرض می‌شود انرژی بیشتری برای شکستن حرارتی پیوند P-F نسبت به پیوند P-Cl مورد نیاز است

**۴. توضیح نقش هالوژن‌ها:**
مکانیسم توضیح می‌دهد چرا جایگزینی همه هالوژن‌ها در (NPCl₂)₃ یا (NPF₂)₃ با گروه‌های آلی فرآیند پلیمریزاسیون را **مسدود می‌کند**:
- القای یونیزاسیون حرارتی یک گروه متیل یا فنیل از فسفر **بسیار دشوار** خواهد بود

#### **مسئله: این داستان کامل نیست!**

**مشاهدات ناسازگار:**

**۱. فسفازن‌های تنش‌دار حلقوی:**
همانطور که قبلاً ذکر شد، فسفازن‌های تنش‌دار حلقوی که فاقد گروه‌های جانبی هالوژن هستند **پلیمریزه می‌شوند**.

**۲. تریمرهای ارگانو-جایگزین‌شده:**
گسترش حلقه را نشان می‌دهند **بدون تشکیل پلیمرهای سنگین**.

**نتیجه:**
- یا دو مکانیسم مستقل درگیر هستند
- یا باید عامل کاملاً دیگری در نظر گرفته شود

#### **عامل دوم مهم: مانع استریک درون‌مولکولی**

**اصل:**
عاملی که به نظر می‌رسد تأثیر قدرتمندی بر توانایی یک فسفازن حلقوی برای پلیمریزه شدن دارد، **مقدار مانع استریک درون‌مولکولی** است که توسط گروه‌های جانبی ایجاد می‌شود.

```

**تعریف:**
سیستم بالای **دمای سقف** خود خواهد بود - دمایی که در آن یک پلیمر همیشه از نظر ترموديناميکی نسبت به گونه‌های مولکول کوچک **ناپایدار** است.

#### **شواهد تجربی برای اثر مانع استریک:**

**حالت معکوس:**
اگر پلیمر را با روش دیگری (مثلاً جایگزینی ماکرومولکولی) بسازیم:
- ممکن است در دماهای متوسط **پایدار** باشد (جایی که سرعت واپلیمریزاسیون بسیار کند است)
- اما وقتی به دماهای بالاتر گرم شود **به تریمر یا تترامر حلقوی واپلیمریزه می‌شود**

**مثال مهم: [NP(OPh)₂]ₙ**
```
[NP(OPh)₂]ₙ:
- در دماهای متوسط: به نظر سینتیکی پایدار شده
- گروه‌های آریلوکسی حجیم: به اندازه کافی پلیمر را از نظر ترموديناميکی ناپایدار می‌کنند
- گرم کردن بالای 150-200°C: واپلیمریزه می‌شود
```

**واکنش:**
```
[NP(OPh)₂]ₙ --[>150-200°C]--> (NPOPh₂)₃ + (NPOPh₂)₄
  پلیمر                           حلقه‌ها
```

**جمع‌بندی عوامل موثر بر پلیمریزاسیون:**

دو عامل اصلی:

**۱. مکانیسم کاتیونی:**
- نیاز به هالوژن‌ها
- مشارکت یون‌ها
- نقش کاتالیزورها

**۲. ترموديناميک/مانع استریک:**
- اندازه گروه‌های جانبی
- دمای سقف
- پایداری نسبی پلیمر vs حلقه

### ۳.۵ پلیمریزاسیون‌های تراکمی که (NPCl₂)ₙ تولید می‌کنند

پلیمریزاسیون باز شدن حلقه (NPCl₂)₃ **تنها راه** برای تولید پلی(دی‌کلروفسفازن) نیست، اگرچه روشی است که **بالاترین وزن‌های مولکولی** پلیمرها را می‌دهد.

برخی **پلیمریزاسیون‌های تراکمی** نیز می‌توانند به این پلیمر منتهی شوند، که سپس به عنوان واسط ماکرومولکولی برای واکنش با آلکوکسیدها، آریلوکسیدها یا آمین‌ها استفاده می‌شود.

**سه فرآیند تراکمی مختلف** شناسایی شده‌اند:

#### **روش ۱: تراکم PCl₅ با آمونیاک**

**پیشینه:**
همانطور که بحث شد، واکنش‌های تراکمی پایه سنتز تریمر حلقوی (NPCl₂)₃ را تشکیل می‌دهند.

**مکانیسم گام‌به‌گام (واکنش ۱۸):**
```
مرحله 1:
PCl₅ + NH₃ → Cl₃P=NH + 2HCl
           (مونومر 3.40)

مرحله 2:
Cl₃P=NH + PCl₅ → Cl₃P=N-PCl₃ + HCl
                 (دایمر خطی 3.41)

مرحله 3:
دایمر + PCl₅ → Cl₃P=N-PCl₂-N=PCl₃ + HCl
                (تریمر خطی 3.42)

و همینطور ادامه...
```

**حلقه‌ای شدن (Cyclization):**
- می‌تواند در هر مرحله بعد از دایمر رخ دهد
- **اما در عمل** احتمال کمتری دارد که زنجیرها فراتر از طول مشخصی رشد کنند

**کاربرد برای پلیمر:**
چندین محقق این فرآیند را برای تولید پلی(دی‌کلروفسفازن) با **وزن مولکولی نسبتاً پایین** گسترش داده‌اند.

**واکنش کلی:**
```
n PCl₅ + n NH₃ → (NPCl₂)ₙ + 4n HCl
                 (وزن مولکولی پایین‌تر)
```

#### **روش ۲: واکنش‌های تراکمی OCl₂PN=PCl₃**

**محققان:** De Jaeger و همکاران

مطالعات گسترده روی پلیمریزاسیون مونومر **OCl₂PN=PCl₃** انجام داده‌اند.

**تهیه مونومر (سه روش):**

**الف) از PCl₅ و سولفات آمونیوم:**
```
PCl₅ + (NH₄)₂SO₄ → OCl₂PN=PCl₃ + محصولات جانبی
```

**ب) از کلرید فسفریل و کلرید آمونیوم:**
```
POCl₃ + NH₄Cl → OCl₂PN=PCl₃ + HCl
```

**ج) از PCl₃ و کلرید آمونیوم تحت شرایط اکسیداتیو:**
```
PCl₃ + NH₄Cl + [O] → OCl₂PN=PCl₃
```

**پلیمریزاسیون (واکنش ۱۹):**
```
شرایط:
- دما: 240-290°C
- فشار: اتمسفری

OCl₂PN=PCl₃ --[گرما]--> (NPCl₂)ₙ + POCl₃
                         حذف کلرید فسفریل
```

**واکنش کلی:**
```
n OCl₂PN=PCl₃ → (NPCl₂)ₙ + n POCl₃
```

**مزایا:**
- **هزینه پایین**

**معایب:**
- **توزیع وزن مولکولی پهن**
- **وزن‌های مولکولی پایین‌تر** نسبت به روش (NPCl₂)₃

**کاربرد:**
با وجود معایب، این فرآیند **به صورت تجاری** استفاده شده است.

#### **روش ۳: پلیمریزاسیون کاتیونی زنده Me₃SiN=PCl₃**

**محققان:** Allcock، Manners و همکاران

**جدیدترین** روش تراکمی برای (NPCl₂)ₙ

**مونومر:** N-سیلیل‌فسفورانیمین
```
Me₃Si-N=PCl₃  (3.43)
```

**ویژگی‌های فرآیند:**
1. پلیمریزاسیون در **فاز حجمی (bulk)** یا **محلول**
2. دمای **اتاق**
3. با حضور مقادیر کم **اسیدهای لوئیس** به عنوان آغازگر

**نتایج:**
- پلیمرها با **توزیع وزن مولکولی بسیار باریک**
- اجازه **کنترل وزن مولکولی** از طریق تغییر نسبت مونومر به آغازگر
- دسترسی به کوپلیمرهای block و comb از پلی‌فسفازن‌ها با پلیمرهای آلی و سیلیکون‌ها

**تهیه مونومر (واکنش ۲۰):**

**روش الف:**
```
Me₃SiCl + PCl₅ + NH₃ → Me₃SiN=PCl₃ + HCl
```

**روش ب:**
```
Me₃SiNH₂ + PCl₅ → Me₃SiN=PCl₃ + 2HCl
```

**روش ج:**
```
(Me₃Si)₂NH + PCl₅ → Me₃SiN=PCl₃ + Me₃SiCl + HCl
```

**مکانیسم پلیمریزاسیون (واکنش‌های ۲۱ و ۲۲):**

**مرحله ۱: آغاز (Initiation) - واکنش ۲۱:**
```
Me₃SiN=PCl₃ + PCl₅ ⇌ Me₃SiCl + [Cl₃P=N-PCl₃]⁺Cl⁻
    مونومر        آغازگر        کاتیون فعال
```

**مرحله ۲: انتشار (Propagation) - واکنش ۲۲:**
```
[Cl₃P=N-PCl₃]⁺ + Me₃SiN=PCl₃ → Cl₃Si + [Cl₃P=N-PCl₂-N=PCl₃]⁺
                                         زنجیر در حال رشد

[—N=PCl₂—N=PCl₃]⁺ + n Me₃SiN=PCl₃ → [—(N=PCl₂)ₙ—N=PCl₃]⁺
   انتهای زنده                          زنجیر بلندتر زنده
```

**ویژگی کلیدی: "Living" Polymerization**

انتهای زنجیر کاتیونی زنده که در واکنش (۲۲) تولید شد ممکن است با افزودن یک فسفورانیمین که دارای یک جایگاه عاملی است **خاتمه یابد**.

**کاربرد مهم:**
این پایه تشکیل کوپلیمرهای block، graft و comb است (بحث بعدی).

**مزایا و معایب:**

**مزایا:**
- توزیع وزن مولکولی باریک
- کنترل دقیق وزن مولکولی
- دسترسی به کوپلیمرها
- دمای واکنش پایین

**معایب:**
- در حال حاضر **گران‌ترین روش**
- هنوز به سطح تجاری **scale-up نشده**

**وضعیت فعلی:**
در مقایسه با روش‌های دیگر، این رویکرد بر بسیاری از نقاط ضعف غلبه می‌کند، اما هنوز برای کاربردهای صنعتی بهینه‌سازی نشده است.

### ۳.۶ پلیمریزاسیون تراکمی فسفورانیمین‌های جایگزین‌شده آلی

#### **اصل:**

پلیمریزاسیون کاتیونی زنده که در بالا توضیح داده شد نیز می‌تواند برای تولید **مستقیم** پلی(ارگانوفسفازن)ها استفاده شود.

**روش:**
```
Me₃SiN=PR₂Cl یا Me₃SiN=PClR₂ + اسید لوئیس → پلیمر
```

**محصولات:**
```
Me₃SiN=PClR₂ → (NPClR)ₙ
Me₃SiN=PR₂Cl → (NPR₂)ₙ
```

**نکته مهم:**
پلیمرهایی که دارای یک کلر و یک گروه آلی در هر فسفر هستند، البته می‌توانند با تیمار با نوکلئوفیل‌های آلی به گونه‌های کاملاً ارگانو-جایگزین‌شده **تبدیل شوند**.

#### **تاریخچه:**

**۱۹۷۷: Flindt، Rose و Marsmann**

اولین پلیمریزاسیون ارگانوفسفورانیمین را گزارش کردند.

**مونومر:**
```
Me₃SiN=P(OCH₂CF₃)₃
   تری‌فلوئورواتوکسی‌دار
```

**فرآیند:**
```
Me₃SiN=P(OCH₂CF₃)₃ --[200°C]--> [NP(OCH₂CF₃)₂]ₙ + Me₃SiOCH₂CF₃
                                  پلیمر        حذف
```

**بهبود بعدی: Matyjaszewski و همکاران**

نشان دادند که پلیمرهای با وزن مولکولی بالاتر را می‌توان از همان مونومر با استفاده از **آغازگرهای آنیونی** مانند Bu₄NF تولید کرد.

#### **سهم اصلی: Wisian-Neilson، Neilson و همکاران**

**کشف کلیدی:**
انواع مختلفی از N-سیلیل‌فسفورانیمین‌های جایگزین‌شده تری‌فلوئورواتوکسی را می‌توان فقط با **گرم کردن در سیستم بسته در ۲۰۰°C** به پلیمرهای سنگین تبدیل کرد.

**فرآیند کلی (واکنش ۲۳):**
```
مرحله 1: تهیه مونومر
RR'PCl + Me₃SiN₃ → Me₃SiN=PRR'Cl + N₂

مرحله 2: پلیمریزاسیون
n Me₃SiN=PRR'Cl --[200°C، بسته]--> (NPRR')ₙ + n Me₃SiCl
```

**مثال‌های مهم:**

شاید **جالب‌ترین** مثال‌ها شامل سنتز:
- `(NPMe₂)ₙ` - پلی(دی‌متیل‌فسفازن)
- `(NPEt₂)ₙ` - پلی(دی‌اتیل‌فسفازن)
- `(NPPr₂)ₙ` - پلی(دی‌پروپیل‌فسفازن)
- **به خصوص** `(NPMePh)ₙ` - پلی(متیل‌فنیل‌فسفازن)

**چرا مهم؟**
اینها پلی(ارگانوفسفازن)هایی هستند که **دشوار یا غیرممکن** است از طریق واکنش‌های جایگزینی ماکرومولکولی از (NPCl₂)ₙ یا (NPF₂)ₙ تهیه شوند.

**کاربرد ویژه (NPMePh)ₙ:**

پلیمر با جانشین‌های مختلط `(NPMePh)ₙ` به طور گسترده توسط این محققان به عنوان بستر برای **جایگزینی‌های ثانویه** از طریق **لیتیایی شدن گروه‌های متیل** استفاده شده است (بحث بعدی).

#### **مشاهده جالب:**

**محدودیت:**
مونومرهایی مانند `Me₃SiN=PClR₂` که یک هالوژن را تحت این شرایط **حذف می‌کنند**، به جای پلیمرهای سنگین، **فسفازن‌های حلقوی کوچک** تولید می‌کنند.

**دلیل:**
```
Me₃SiN=PClR₂ --[گرما]--> حلقه‌های (NPR₂)₃,₄ + Me₃SiCl
                          نه پلیمر سنگین!
```

### ۳.۷ فرآیند تراکم آزید

**رویکرد جایگزین:**
نوع دیگری از سنتز تراکمی شامل تجزیه **آزیدهای ارگانوفسفر** است.

**محققان اولیه:** Haber و همکاران (دهه ۱۹۶۰)

**مسیر (واکنش ۲۴):**
```
R₂P-N₃ --[محلول، گرما]--> حذف N₂
         ↓
    (NPR₂)₃ یا (NPR₂)ₙ
    حلقه      پلیمر
```

**نکات واکنش:**
- حذف نیتروژن در محلول انجام می‌شود
- تشکیل مستقیم یک فسفازن حلقوی یا پلیمر

#### **کاربرد: Matyjaszewski و همکاران**

این روش را برای تولید استفاده کرده‌اند:

**۱. (NPPh₂)ₙ:**
```
Ph₂P-N₃ --[محلول]--> (NPPh₂)ₙ
                      تقریباً نامحلول
```
- **دشوار است** با روش‌های دیگر تهیه شود
- تقریباً نامحلول

**۲. [NP(Ph)(p-MePh)]ₙ:**
```
Ph(p-MeC₆H₄)P-N₃ → [NP(Ph)(p-MePh)]ₙ
                    محلول در حلال‌های آروماتیک
```
- محلول در حلال‌های آروماتیک
- گزینه بهتر از (NPPh₂)ₙ

#### **هشدار ایمنی مهم! ⚠️**

**آزیدهای فسفر مواد بالقوه منفجره هستند!**

**احتیاط‌های لازم:**
- باید در **محلول** تهیه و استفاده شوند
- **هرگز** در حالت جامد جداسازی نشوند
- می‌توانند به عنوان دتوناتور عمل کنند

### ۳.۸ جایگزینی‌های ماکرومولکولی ثانویه

#### **مفهوم:**

فرآیندهای سنتز اولیه پلی‌فسفازن که تاکنون بحث شد، دسترسی به **انواع عظیمی** از ماکرومولکول‌های مختلف را فراهم می‌کنند.

**اما:**
تنوع ساختاری بیشتر را می‌توان با انجام **واکنش‌های اصلاحی** روی پلیمرهای ارگانوفسفازن تولید شده در این واکنش‌های اولیه به دست آورد.

**اهمیت ویژه:**
واکنش‌های ثانویه به خصوص برای تهیه پلیمرها با **واحدهای عاملی** در ساختار گروه جانبی مهم هستند - گونه‌هایی که **دشوار یا غیرممکن** است با روش‌های سنتز اولیه تولید شوند.

**دامنه:**
**بسیاری** از واکنش‌های جایگزینی ثانویه مختلف گزارش شده‌اند.

#### **مثال‌های نماینده:**

**۱. تبدیل‌های آلی روی حلقه‌های آریل**

یکی از پرکاربردترین‌ها تبدیل‌های آلی روی **حلقه‌های آریل** است که بخشی از ساختار گروه جانبی را تشکیل می‌دهند.

**مثال الف: سولفوناسیون پلی(دی‌فنوکسی‌فسفازن)**
```
[NP(OPh)₂]ₙ + SO₃ → [NP(OC₆H₄SO₃H)₂]ₙ
             یا H₂SO₄

واکنش (۲۵):
[NP(O-C₆H₅)₂]ₙ → [NP(O-C₆H₄-SO₃H)₂]ₙ
```

**مثال ب: نیتراسیون و کاهش**
```
[NP(OPh)₂]ₙ + HNO₃ → [NP(OC₆H₄NO₂)₂]ₙ
                      نیتروفنوکسی

[NP(OC₆H₄NO₂)₂]ₙ + [H] → [NP(OC₆H₄NH₂)₂]ₙ
                          آمینوفنوکسی

واکنش (۲۶):
[NP(O-C₆H₅)₂]ₙ → [نیترا] → [NP(O-C₆H₄-NO₂)₂]ₙ
                           ↓ [کاهش]
                  [NP(O-C₆H₄-NH₂)₂]ₙ
```

**مثال ج: لیتیایی کردن پلیمرهای برموفنوکسی**
```
[NP(OC₆H₄Br)₂]ₙ + nBuLi → [NP(OC₆H₄Li)₂]ₙ
                           جایگاه لیتیایی

سپس واکنش با:
- R₂PCl → واحدهای فسفین
- R₃SiCl → واحدهای ارگانوسیلیکون
- MLₓ → واحدهای ارگانوفلزی
- CO₂ → واحدهای اسید کربوکسیلیک

واکنش (۲۷):
[NP(O-C₆H₄-Br)₂]ₙ + Li → [NP(O-C₆H₄-Li)₂]ₙ
                                ↓ + R₂PCl
                       [NP(O-C₶H₄-PR₂)₂]ₙ
```

**مثال د: لیتیایی کردن گروه‌های جانبی متیل**
```
[NP(Me)(Ph)]ₙ + nBuLi → [NP(CH₂Li)(Ph)]ₙ
                        ↓ + R₃SiCl یا MLₓ
                واحدهای ارگانوسیلیکون یا ارگانوفلزی

واکنش (۲۸):
[NP(CH₃)(Ph)]ₙ → [لیتیایی] → [NP(CH₂Li)(Ph)]ₙ
                              ↓ + واکنش‌دهنده
                    پلیمر اصلاح‌شده
```

**۲. هیدرولیز استرهای اسید کربوکسیلیک**
```
[NP(OC₆H₄COOEt)₂]ₙ + H₂O/OH⁻ → [NP(OC₆H₄COOH)₂]ₙ
    استر محافظت‌شده                 اسید آزاد
```

**۳. واکنش‌های deprotection**
```
انواع مختلفی از گروه‌های آمینو و الکل محافظت‌شده در انتهای
گروه‌های جانبی آروماتیک و آلیفاتیک → واکنش‌های deprotection
→ گروه‌های عاملی آمینو یا الکل آزاد
```

#### **واکنش‌های نمونه (۲۵-۲۸) با جزئیات بیشتر:**

**واکنش ۲۵ - سولفوناسیون:**
```
[NP(O-C₆H₅)₂]ₙ + SO₃ → 

        OSO₃H          OSO₃H
         |              |
[—N═P—N═P—]ₙ
         |              |
        O              O
         |              |
        C₆H₅          C₆H₄SO₃H
```

**واکنش ۲۶ - نیتراسیون/کاهش:**
```
[NP(OPh)₂]ₙ --[HNO₃/H₂SO₄]--> [NP(OC₆H₄NO₂)₂]ₙ

[NP(OC₆H₄NO₂)₂]ₙ --[Na₂S₂O₄]--> [NP(OC₆H₄NH₂)₂]ₙ

## 🔬 بخش چهارم: واکنش‌های سطحی پلی‌فسفازن‌ها

### ۴.۱ مفهوم و اهمیت

#### **مشکل عمومی در علم مواد:**

برخی پلیمرها **مجموعه خاصی از خواص حجمی (bulk)** دارند که آنها را برای یک کاربرد خاص ایده‌آل می‌کند، اما **نمی‌توانند استفاده شوند** چون **خواص سطحی نامناسب** هستند.

**مثال‌ها:**
1. **ماده‌ای با کشسانی عالی** اما نمی‌تواند در دستگاه‌های قلبی-عروقی استفاده شود چون **سطح پلیمر لخته شدن خون را تحریک می‌کند**
2. **پلیمر دیگری** سازگاری زیستی سطحی عالی دارد اما **برای کاربرد قلبی-عروقی خیلی شکننده است**

**راه حل:**
انتخاب پلیمر برای **خواص حجمی مطلوب** آن و سپس انجام **واکنش‌های اصلاحی خاصیت روی سطح پلیمر** بدون تأثیر بر ماده حجمی.

### ۴.۲ چرا پلی‌فسفازن‌ها برای شیمی سطحی مناسب هستند؟

**مزایای ویژه:**

**۱. تنوع گروه‌های جانبی اولیه:**
انواع گسترده‌ای از گروه‌های جانبی آلی اولیه را می‌توان استفاده کرد

**۲. قابلیت تبدیل آلی:**
این گروه‌ها می‌توانند تحت انواع مختلفی از تبدیل‌های آلی قرار گیرند

**۳. محافظت اسکلت:**
**نکته مهم:** در بیشتر موارد زنجیره پلیمری توسط گروه‌های جانبی از واکنش‌دهنده‌های تهاجمی **محافظت می‌شود**

**نتیجه:**
انواع گسترده‌ای از واکنش‌های سطحی بررسی شده‌اند برای بهینه‌سازی پلیمرهای مختلف برای کاربردها در:
- پزشکی
- پوشش‌های سطحی
- چسبندگی
- مقاومت در برابر آب و هوا
- تثبیت (immobilization) گونه‌هایی مانند آنزیم‌ها یا مولکول‌های فعال زیستی دیگر

### ۴.۳ واکنش‌های سطحی نمونه

**واکنش‌های ۲۹-۳۳ چند اصلاح سطحی که بررسی شده‌اند را خلاصه می‌کنند:**

#### **واکنش ۲۹: تبدیل تری‌فلوئورواتوکسی به هیدروکسیل**
```
سطح: —N═P(OCH₂CF₃)₂— + NaOH/Bu₄NBr (آبی) 
           ↓
      —N═P(OH)₂—
      
تبدیل سطح آبگریز → سطح آبدوست
```

**جزئیات:**
```
        OCH₂CF₃  OCH₂CF₃         OH      OH
         |        |               |       |
سطح: —N═P——N═P—     →    —N═P——N═P—
         |        |               |       |
        OCH₂CF₃  OCH₂CF₃         OH      OH

آبگریز (زاویه تماس ~100°) → آبدوست (زاویه تماس ~0°)
```

**کاربرد:**
- تبدیل سطح غیرچسبنده، بسیار آبگریز به سطح چسبنده آبدوست
- تغییرات در شرایط واکنش اجازه می‌دهد:
  - عمق تبدیل کنترل شود
  - نسبت گروه‌های سطحی اولیه به جدید کنترل شود

#### **واکنش ۳۰: افزودن آب به پیوندهای دوگانه سطحی**
```
سطح با گروه‌های وینیل: —N═P—CH₂—CH═CH₂
                               ↓ + H₂O
سطح هیدروکسیل‌دار: —N═P—CH₂—CH(OH)—CH₃
```

#### **واکنش ۳۱: تبدیل فنوکسی به نیترو سپس آمینو**
```
[NP(OPh)₂]ₙ سطح → [نیتراسیون] → [NP(OC₆H₄NO₂)₂]ₙ سطح
                                  ↓ [کاهش]
                        [NP(OC₆H₄NH₂)₂]ₙ سطح

جزئیات:
        OPh   OPh               O-C₆H₄-NO₂
         |     |                 |
سطح: —N═P—N═P—   + HNO₃  →  —N═P—
         |     |                 |
        OPh   OPh               O-C₆H₄-NO₂

سپس + [H]:  O-C₆H₄-NH₂
              |
          —N═P—
              |
            O-C₆H₄-NH₂
```

**کاربرد:**
گروه‌های آمینو سطحی برای اتصال بیشتر مولکول‌های زیستی استفاده می‌شوند.

#### **واکنش ۳۲: سولفوناسیون سطح**
```
[NP(OPh)₂]ₙ سطح + H₂SO₄ (غلیظ) → [NP(OC₆H₄SO₃H)₂]ₙ سطح

        O-Ph                    O-C₆H₄-SO₃H
         |                       |
سطح: —N═P—      →           —N═P—
         |                       |
        O-Ph                    O-C₆H₄-SO₃H

آبگریز (زاویه تماس 101°) → آبدوست (زاویه تماس 0°)
```

**نکته بسیار مهم:**
شرایط واکنش باید **با دقت کنترل شود** تا از انحلال پلیمر در اسید قوی و شکستن زنجیر بعدی جلوگیری شود.

**تحت شرایط مناسب:**
یک لایه بیرونی از واحدهای آریلوکسی‌فسفازن سولفونه‌شده تشکیل می‌شود.

**نتیجه:**
این تغییر یک سطح آبگریز (زاویه تماس ۱۰۱°) را به سطح آبدوست (زاویه تماس ۰°) در **یک مرحله ساده** تبدیل می‌کند.

**تأثیر بر سازگاری زیستی:**
تأثیر این تبدیل بر سازگاری خون و بافت هنوز در حال بررسی است، اما شناخته شده است که تغییرات مشابه در خاصیت سطحی پلیمرهای آلی **بهبود چشمگیری** در سازگاری خون به وجود می‌آورد.

#### **واکنش ۳۳: تثبیت هپارین روی سطح**
```
سطح—NH₂ + هپارین (ضدانعقاد خون) → سطح—NH—هپارین

یا

سطح—OH + هپارین → سطح—O—هپارین
```

**نتیجه:**
تثبیت هپارین **افزایش پنج برابری** در زمان انعقاد خون به وجود آورد.

#### **مثال دیگر: تثبیت دوپامین**
```
سطح + دوپامین → سطح—دوپامین

نتیجه: دوپامین تثبیت‌شده همان پاسخ را در سلول‌های
هیپوفیز موش ایجاد کرد که دوپامین آزاد
```

### ۴.۴ کاربردهای اصلاح سطح

**۱. راکتورهای بیوشیمیایی:**
پلیمرهای عامل‌دار شده سطحی برای کاربردها در راکتورهای بیوشیمیایی و حسگرهای زیست‌پزشکی مورد توجه هستند.

**۲. تثبیت آنزیم:**
تثبیت آنزیم‌ها روی سطح پلیمر - مثال مهم

**دلایل تثبیت آنزیم:**

**الف) افزایش طول عمر:**
تثبیت اغلب مدت زمانی را که پروتئین فعالیت کاتالیزوری خود را حفظ می‌کند **افزایش می‌دهد**، در مقایسه با همان آنزیم آزاد در محلول.

**ب) جداسازی آسان:**
تثبیت **جداسازی آنزیم از محصولات** واکنش القا شده توسط آنزیم را تسهیل می‌کند.
- **مثال:** استفاده از کونژوگه پلیمر/آنزیم به عنوان فاز ثابت در یک راکتور جریان پیوسته

**ج) سیستم‌های چند آنزیمی:**
ممکن است بتوان چندین آنزیم مختلف را روی همان سطح تثبیت کرد:
```
سطح—آنزیم₁—آنزیم₂—آنزیم₃

محصولات واکنش آنزیم₁ → مواد اولیه آنزیم₂ → و غیره
```

**د) سطح بالا:**
برای همه چنین فرآیندهایی معمولاً مطلوب است از شکلی از پلیمر تثبیت که **بیشترین سطح ممکن** را فراهم می‌کند استفاده شود.
- **بنابراین:** معمولاً ذرات کوچک پلیمر استفاده می‌شوند

### ۴.۵ مثال تفصیلی: تثبیت آنزیم روی پلی‌فسفازن

#### **سیستم مورد استفاده:**

**پلیمر پایه:** `[NP(OPh)₂]ₙ` - پلی[بیس(فنوکسی)فسفازن]

**استراتژی برای افزایش سطح:**
رسوب یک **فیلم نازک** از `[NP(OPh)₂]ₙ` روی سطح ذرات کوچک **بسیار متخلخل** پودر آلومینا.

**نکته کلیدی:**
اگر رسوب پلیمر **با دقت کنترل شود** تا منافذ میکروسکوپی را **مسدود نکند**، می‌توان سطح عظیمی از پلیمر تولید کرد (شکل ۳.۱۹).

**توضیح شکل ۳.۱۹:**
میکروگراف الکترونی روبشی از سطح یک ذره آلومینای متخلخل پوشیده شده با `[NP(OPh)₂]ₙ`.

#### **دنباله واکنش (واکنش ۵۶):**
```
مرحله 1: نیتراسیون سطحی
[NP(O-Ph)₂]ₙ سطح + HNO₃/H₂SO₄ → [NP(O-C₆H₄-NO₂)₂]ₙ سطح
                                   واحدهای نیتروفنوکسی در سطح

مرحله 2: کاهش
[NP(O-C₆H₄-NO₂)₂]ₙ سطح + Na₂S₂O₄ → [NP(O-C₆H₄-NH₂)₂]ₙ سطح
                                      عملکردهای آمینو در سطح

مرحله 3: اتصال آنزیم
در این مرحله انواع مختلفی از روش‌ها برای اتصال آنزیم‌ها
استفاده شد:
- تریپسین
- گلوکز 6-فسفات دهیدروژناز
```

**روش اتصال موثرترین:**
استفاده از **گلوتاریک دی‌آلدهید** [OHC(CH₂)₃CHO] به عنوان عامل اتصال‌دهنده

**مکانیسم:**
```
سطح—NH₂ + OHC—(CH₂)₃—CHO + H₂N—آنزیم
                     ↓
    سطح—N═CH—(CH₂)₃—CH═N—آنزیم
        (پیوند شیف بیس)
```

این واکنش‌دهنده با **هر دو** واکنش می‌دهد:
1. گروه‌های آمینو روی سطح
2. گروه‌های آمینو روی آنزیم

→ فرآیند تثبیت را کامل می‌کند

#### **نتایج (شکل ۳.۲۰):**

**بخش بزرگی از فعالیت آنزیمی پس از تثبیت حفظ شد:**

نمودارهایی که فعالیت آنزیم گلوکز-۶-فسفات دهیدروژناز را به عنوان تابعی از زمان نشان می‌دهند:
- ⚪ دایره‌های باز: آنزیم آزاد در محلول
- ⚫ دایره‌های سیاه: آنزیم متصل به سطح `[NP(OPh)₂]ₙ`

**راکتور جریان پیوسته:**
علاوه بر این، بسته‌بندی ذرات کونژوگه در یک ستون نوع کروماتوگرافی اجازه **ساخت راکتور جریان پیوسته** را داد.

**مشاهده مهم:**
شکل ۳.۲۰ نشان می‌دهد که مولکول‌های آنزیم برای **مدت طولانی** به کاتالیز واکنش‌های خود ادامه دادند.

#### **چرا پلی‌فسفازن سطح عالی است؟**

**۱. پایداری اسکلت:**
پلی‌فسفازن یک سطح بستر عالی است به دلیل **پایداری زنجیره پلیمری** در برابر نیتراسیون و کاهش.

**۲. قابلیت تنظیم آبگریزی/آبدوستی:**
سهولتی که با آن می‌توان **آبگریزی یا آبدوستی سطح** را تغییر داد.

#### **کاربردهای عملی آنزیم‌های تثبیت‌شده:**

**۱. صنایع بیوشیمیایی و تخمیر**
**۲. تجهیزات تشخیص پزشکی**
**۳. احتمالاً در ساخت کبد مصنوعی**

### ۴.۶ نکات مهم درباره واکنش‌های سطحی

#### **تغییرات هیدروفوب ↔ هیدروفیل:**

**مشاهده:**
توجه کنید که بسیاری از این واکنش‌های سطحی شامل **تبدیل یک پلیمر آبگریز به پلیمری با سطح آبدوست است یا برعکس**.

**مثال:**
جایگزینی گروه‌های تری‌فلوئورواتوکسی در سطح با واحدهای هیدروکسیل:
```
—OCH₂CF₃ (آبگریز، غیرچسبنده) → —OH (آبدوست، چسبنده)
```

یک سطح غیرچسبنده، بسیار آبگریز را به یک سطح چسبنده آبدوست تغییر می‌دهد.

#### **کنترل پارامترها:**

تغییرات در شرایط واکنش اجازه می‌دهند که **هر دو** کنترل شوند:
1. **عمق تبدیل**
2. **نسبت گروه‌های سطحی اولیه به گروه‌های جدید**

#### **عارضه احتمالی: حرکات مولکولی پلیمر**

**مسئله:**
یک عارضه احتمالی که باید برای **همه** این تبدیل‌های سطحی در نظر گرفته شود این است که **حرکات مولکولی پلیمر** ممکن است واحدهای عاملی تازه معرفی‌شده را **دفن کنند** اگر پلیمر با محیط‌های خاصی تماس پیدا کند.

**مثال:**
```
سطح آبدوست روی پلیمر آبگریز:

در تماس با هوای خشک یا مایع آبگریز:
→ سطح آبدوست دفن می‌شود (واحدها به داخل می‌روند)

در تماس با مایع آبدوست:
→ واحدهای آبدوست دوباره به سطح می‌آیند
```

**نتیجه:**
این فرآیند **برگشت‌پذیر** است - قرار گرفتن در معرض یک مایع آبدوست می‌تواند این فرآیند را معکوس کند.

## 📊 بخش پنجم: سیستم‌های هیبریدی از طریق کوپلیمرهای Block، Comb یا Ring-Linked

### ۵.۱ دامنه و اهمیت

#### **مفهوم هیبریدسازی:**

روش‌هایی اکنون برای **هیبریدسازی فسفازن‌ها با سیستم‌های پلیمری دیگر** در دسترس هستند تا خواص ویژه ماکرومولکول‌های معدنی را با خواص پلیمرهای آلی کلاسیک یا پلی(ارگانوسیلوکسان)ها **ترکیب کنند**.

#### **مزایای احتمالی سیستم‌های هیبریدی:**

**۱. کاهش هزینه:**
کاهش هزینه یک ماده

**۲. خواص ویژه فسفازن‌ها:**
استفاده از خواص ویژه فسفازن‌ها مانند:
- مقاومت در برابر آتش
- آب‌گریزی
- سهولت اتصال عوامل زیستی
- و غیره

### ۵.۲ روش‌های هیبریدسازی

هیبریدسازی را می‌توان **به چند روش مختلف** انجام داد:

#### **روش ۱: کوپلیمرهای Block**

**تولید با استفاده از پلیمریزاسیون کاتیونی زنده** (که قبلاً بحث شد):
- تبدیل پلیمرهای زنده به ماکرومولکول‌های عامل‌دار انتهایی یا تله‌کلیک (telechelic)

**ساختار:**
```
—[بلوک A]—[بلوک B]— (دی‌بلوک)
—[بلوک A]—[بلوک B]—[بلوک A]— (تری‌بلوک)

A = پلی‌فسفازن
B = پلیمر آلی یا سیلیکون
```

#### **روش ۲: کوپلیمرهای Comb یا Graft**

پلی‌فسفازن‌ها را می‌توان با **گروه‌های وینیل، آلیل یا نوربورننیل** در انتهای زنجیر تهیه کرد.

**سپس:**
این گروه‌ها را می‌توان **پلیمریزه یا کوپلیمریزه** کرد با مونومرهای غیراشباع آلی:
```
پلیمر آلی اصلی
    |
    زنجیر فسفازن (شانه)
    |
    زنجیر فسفازن
    |
    زنجیر فسفازن
```

→ پلیمرهای آلی با زنجیرهای جانبی پلی‌فسفازن معلق (کوپلیمرهای comb یا graft)

#### **روش ۳: پلیمرهای آلی با حلقه‌های سیکلوفسفازن جانبی**

حلقه‌های فسفازن مولکول کوچک را می‌توان با واحدهای جانبی وینیل، آلیل یا نوربورننیل عامل‌دار کرد.

**سپس:**
پلیمریزاسیون این گروه‌ها:
```
—CH₂—CH—CH₂—CH—CH₂—CH—
       |        |        |
      حلقه    حلقه    حلقه
     فسفازن  فسفازن  فسفازن
```

→ پلیمرهای آلی با گروه‌های جانبی سیکلوفسفازن

#### **روش ۴: مواد Cyclolinear یا Cyclomatrix**

حلقه‌های فسفازن را می‌توان از طریق واحدهای پیوند‌دهنده پلیمر آلی یا الیگومر **به یکدیگر متصل کرد**:

**Cyclolinear:**
```
حلقه—لینکر—حلقه—لینکر—حلقه—لینکر—
```

**Cyclomatrix:**
```
حلقه———حلقه
 |   \ /   |
لینکر  ×  لینکر
 |   / \   |
حلقه———حلقه
```

### ۵.۳ کوپلیمرهای Block (جزئیات)

#### **اصل:**

پلیمریزاسیون کاتیونی زنده N-سیلیل‌فسفورانیمین‌ها (بحث شده در بخش ۳.۳.۵) ماکرومولکول‌های "زنده" تولید می‌کند که با **خاتمه‌دهنده‌های فسفورانیمین** واکنش می‌دهند.

**جایگزین:**
همان مولکول‌هایی که به عنوان خاتمه‌دهنده عمل می‌کنند می‌توانند به عنوان **آغازگرهایی** استفاده شوند که از آنها زنجیرهای پلی‌فسفازن می‌توانند **رشد کنند**.

#### **ساختار خاتمه‌دهنده‌ها/آغازگرها:**

فرمول کلی: **Me₃SiN=PR₂R'**

جایی که:
- **R** = گروه آلی غیرواکنشی (مثل تری‌فلوئورواتوکسی)
- **R'** = واحدی که دارای عملکرد است

**مثال‌های ساختار۳.۴۴-۳.۴۹:**

3.44: Me₃SiN=P(OCH₂CF₃)₂—CH₂—CH═CH₂
(گروه آلیل)

3.45: Me₃SiN=P(OCH₂CF₃)₂—Ph—CH═CH₂
(گروه استایرنیل)

3.46: Me₃SiN=P(OCH₂CF₃)₂—(CH₂)ₙ—CH═CH₂
(زنجیر آلیفاتیک با وینیل)

3.47: Me₃SiN=P(OCH₂CF₃)₂—نوربورنن
(گروه نوربورننیل)

3.48: Me₃SiN=P(OCH₂CF₃)₂—

Continue

(CH₂)ₙ—OH
(گروه هیدروکسیل)

3.49: Me₃SiN=P(OCH₂CF₃)₂—(CH₂)ₙ—COOH
(گروه کربوکسیلیک اسید)


**شرط مهم:**
به شرطی که **R' یک اتم هالوژن X نباشد** (که به صورت Me₃SiX حذف می‌شود)، خاتمه زنجیر پلی‌فسفازن **به جای پلیمریزاسیون بیشتر اتفاق می‌افتد**.

#### **توالی واکنش (۳۴):**
```
مرحله 1: آغاز پلیمریزاسیون
PCl₅ + Me₃SiN=PCl₃ → [—N=PCl₂—]ₙ⁺
                      زنجیر زنده

مرحله 2: خاتمه با فسفورانیمین عاملی
[—N=PCl₂—]ₙ⁺ + Me₃SiN=P(OCH₂CF₃)₂—R' 
                     ↓
        [—N=PCl₂—]ₙ—N=P(OCH₂CF₃)₂—R'
         پلیمر با گروه عاملی انتهایی

مرحله 3: جایگزینی کلرها
[—N=PCl₂—]ₙ—N=P(OCH₂CF₃)₂—R' + NaOR"
                     ↓
     [—N=P(OR")₂—]ₙ—N=P(OCH₂CF₃)₂—R'
      پلیمر عاملی‌دار آماده برای اتصال
```

**نکته بسیار مهم:**
پس از اینکه پلیمریزاسیون توسط خاتمه‌دهنده فسفورانیمین خاتمه یافت، واحدهای جانبی کلر در طول زنجیر پلی‌فسفازن **باید** با گروه‌های آلی غیرواکنشی جایگزین شوند **قبل از** اینکه به مرحله پیوند انتهای پلیمر بروید.

#### **انواع ساختار بسته به نسبت اولیه:**

**نکته:** بسته به نسبت اولیه PCl₅ به مونومر:

**حالت 1: یک انتهای زنده**
```
PCl₅ + مونومر (زیاد) → P—[پلیمر]⁺
                        یک انتهای زنده
```

**حالت 2: هر دو انتها زنده**
```
PCl₅ (زیاد) + مونومر → ⁺[پلیمر]⁺
                       دو انتهای زنده
```

**بنابراین:**
فرآیند خاتمه می‌تواند ساختاری را فراهم کند که برای **پیوند به پلیمر دیگر** از طریق:
- یک انتها → **کوپلیمرهای دی‌بلوک**
- دو انتها → **کوپلیمرهای تری‌بلوک**

**ساختارهای عمومی:**
```
3.12 (دی‌بلوک):
[پلی‌فسفازن]—Sw—[پلیمر آلی]

3.13 (تری‌بلوک A-B-A):
[پلیمر آلی]—Sw—[پلی‌فسفازن]—Sw—[پلیمر آلی]

3.14 (تری‌بلوک B-A-B):
[پلی‌فسفازن]—Sw—[پلیمر آلی]—Sw—[پلی‌فسفازن]

Sw = گروه "switchover" (جایگاه اتصال)
```

#### **روش جایگزین (واکنش ۳۵):**

**استراتژی:**
یکی از مولکول‌های آغازگر (مثل ۳.۴۴) را می‌توان به **انتهای یک پلیمر آلی** متصل کرد و سپس برای **آغاز رشد** یک زنجیر پلی(دی‌کلروفسفازن) استفاده کرد.

**توالی:**
```
مرحله 1: اتصال آغازگر به پلیمر آلی
پلیمر آلی—OH + Cl—Si(CH₃)₂—N=P(OCH₂CF₃)₂Cl
                     ↓
    پلیمر آلی—O—Si(CH₃)₂—N=P(OCH₂CF₃)₂Cl
           ماکروآغازگر

مرحله 2: پلیمریزاسیون
ماکروآغازگر + Me₃SiN=PCl₃ + PCl₅
                     ↓
پلیمر آلی—O—Si(CH₃)₂—[N=PCl₂]ₙ⁺
                زنجیر زنده فسفازن

مرحله 3: خاتمه و جایگزینی
→ پلیمر آلی—Sw—[پلی‌فسفازن]
```

#### **نمونه‌های کوپلیمرهای Block (۳.۵۰-۳.۵۳):**

**۳.۵۰:**
```
[پلی‌فسفازن]—Sw—[پلی(اتیلن اکساید)]
```

**۳.۵۱:**
```
[پلی‌فسفازن]—Sw—[پلی‌استایرن]
```

**۳.۵۲:**
```
[پلی(دی‌متیل‌سیلوکسان)]—Sw—[پلی‌فسفازن]—Sw—[پلی(دی‌متیل‌سیلوکسان)]
```

**۳.۵۳:**
```
[پلی‌متاکریلات]—Sw—[پلی‌فسفازن]
```

**در این ساختارها Sw به گروه "switchover" اشاره دارد.**

#### **خواص و کاربردها:**

کوپلیمرهای Block که حاوی بخش‌هایی با ویژگی‌های متفاوت هستند، مانند:
- بلوک‌های آبگریز و آبدوست

**می‌توانند:**
1. سیستم‌های جامد **جدا شده فازی** تشکیل دهند
2. به کار رفته شوند برای تشکیل **میسل‌ها یا وزیکول‌ها** در محیط آبی

**فرصت‌های طراحی:**
با در نظر گرفتن اینکه طول هر بلوک پلیمری **قابل کنترل** است، فرصت‌ها برای تغییرات خاصیت از طریق چنین سیستم‌هایی **تقریباً نامحدود** هستند.

### ۵.۴ کوپلیمرهای Comb و Graft

#### **مفهوم (ساختار ۳.۱۵):**
```
        |             |             |
    —پلیمر اصلی——پلیمر اصلی——پلیمر اصلی—
        |             |             |
   زنجیر        زنجیر        زنجیر
  فسفازن      فسفازن      فسفازن
  معلق        معلق        معلق
```

پلیمرهای Comb یا graft از طریق پلیمریزاسیون واحدهای وینیل، آلیل یا نوربورننیل که **کووالانسی به انتهای زنجیرهای پلی‌فسفازن** متصل شده‌اند، **قابل دسترسی** هستند.

#### **روش‌های سنتز:**

**روش 1: از طریق خاتمه زنجیر زنده**

همانطور که قبلاً ذکر شد، این را می‌توان از طریق **خاتمه یک زنجیر پلی‌فسفازن زنده** توسط یک فسفورانیمین که یکی از این واحدها را دارد انجام داد.

**سپس:**
کوپلیمر comb با:
- **پلیمریزاسیون رادیکال آزاد** گروه‌های عاملی
- **پلیمریزاسیون متاتز باز شدن حلقه (ROMP)**

**یا:**
کوپلیمریزاسیون با مونومرهای آلی می‌تواند کوپلیمرهای graft تولید کند.

#### **مثال‌ها:**

**واکنش ۳۶: Comb از طریق پلیمریزاسیون رادیکالی**
```
مرحله 1: تهیه پلی‌فسفازن با گروه‌های استایرنیل انتهایی
PCl₅ + Me₃SiN=PCl₃ → [پلیمریزاسیون زنده]
    → [—N=PCl₂—]ₙ⁺

[—N=PCl₂—]ₙ⁺ + Me₃SiN=P(OCH₂CF₃)₂—Ph—CH═CH₂
    → [—N=PCl₂—]ₙ—N=P(OCH₂CF₃)₂—Ph—CH═CH₂

    → [جایگزینی با NaOCH₂CF₃]
    → [—N=P(OCH₂CF₃)₂—]ₙ—N=P(OCH₂CF₃)₂—Ph—CH═CH₂
      پلیمر با انتهای استایرنیل

مرحله 2: پلیمریزاسیون رادیکالی
[پلی‌فسفازن]—Ph—CH═CH₂ + مونومر وینیلی
    → [پلیمریزاسیون رادیکال آزاد]
    → [پلی‌فسفازن]—Ph—CH₂—CH₂—[پلیمر وینیلی]
```

**واکنش ۳۷: Comb از طریق ROMP**
```
مرحله 1: پلی‌فسفازن با گروه نوربورننیل
مشابه بالا اما با انتهای نوربورننیل:
[—N=P(OCH₂CF₃)₂—]ₙ—نوربورنن

مرحله 2: پلیمریزاسیون ROMP
[پلی‌فسفازن]—نوربورنن + کاتالیزور ROMP
    → [پلی‌فسفازن]—[پلی‌نوربورنن]
                    یا
    + نوربورنن → [پلی‌فسفازن]—[کوپلیمر نوربورنن]
```

#### **روش جایگزین: فتولیز یا رادیولیز**

پلیمرهای آلی با graft های پلی‌فسفازن یا پلی‌فسفازن‌ها با graft های پلیمر آلی نیز می‌توانند با **فتولیز یا رادیولیز** مخلوط دو نوع پلیمر تولید شوند.

**روش:**
```
پلیمر A + پلیمر B → [تابش UV یا گاما]
                   → رادیکال‌ها تشکیل می‌شوند
                   → اتصال زنجیرها
                   → کوپلیمر graft
```

### ۵.۵ پلیمرهای آلی با حلقه‌های فسفازن به عنوان واحدهای جانبی

#### **مفهوم و اهمیت:**

حلقه تریمری سیکلوفسفازن یک **واحد پایدار** با خواص تکنولوژیکی ارزشمند است، مانند:
- مقاومت در برابر گرما
- رفتار بازدارنده شعله

**نتیجه:**
پلیمرهای آلی که حاوی فسفازن‌های مولکول کوچک هستند **قابلیت اشتعال بسیار کمتری** نسبت به نمونه‌های مشابه که فسفازن در آنها وجود ندارد دارند.

#### **رویکردهای قدیمی:**

**Rayon مقاوم در برابر آتش:**
یک Rayon (سلولز بازسازی شده) مقاوم در برابر آتش ساخته شده که حاوی آلکوکسی‌سیکلوفسفازن‌ها است که به صورت **فیزیکی در فیبرهای پلیمر آلی به دام افتاده‌اند**.

**محدودیت:**
برای بسیاری از کاربردها مفیدتر است که حلقه‌های فسفازن **کووالانسی به یک زنجیر پلیمر آلی متصل** باشند نه اینکه صرفاً در این ماده به دام افتاده باشند.

**دلیل:**
به این ترتیب یک فسفازن مولکول کوچک بازدارنده آتش نمی‌تواند با **نفوذ در طول زمان طولانی** فرار کند.

#### **سه روش توسعه یافته:**

**روش 1: حلقه‌های فسفازن با یک گروه وینیل یا آلیل**

حلقه‌های سیکلوفسفازن سنتز شده‌اند که دارای **یک گروه وینیل یا آلیل در هر حلقه** هستند.

**مونومر نمونه (۳.۵۴) - واکنش ۳۸:**
```
        OCH₂CF₃  OCH₂CF₃
         |        |
    (CF₃CH₂O)P—N—N—P(OCH₂CF₃)
              |     |
              N     N
               \   /
                P
               / \
         OCH₂CF₃  O—(CH₂)ₙ—CH═CH₂

مونومر 3.54

پلیمریزاسیون:
n (مونومر 3.54) → [پلیمریزاسیون رادیکالی]

—CH₂—CH—CH₂—CH—CH₂—CH—
       |        |        |
     حلقه    حلقه    حلقه
    فسفازن  فسفازن  فسفازن
```

**روش 2: گروه‌های نوربورننیل متصل به حلقه‌های فسفازن**

گروه‌های نوربورننیل را می‌توان به حلقه‌های فسفازن متصل کرد، همانطور که در **۳.۵۵** نشان داده شده است.

**واکنش ۳۹:**
```
مونومر 3.55:
حلقه فسفازن—O—(CH₂)ₙ—نوربورنن

پلیمریزاسیون ROMP:
n (مونومر 3.55) + کاتالیزور Grubbs
    → [پلی‌نوربورنن با حلقه‌های فسفازن معلق]

یا کوپلیمریزاسیون:
(مونومر 3.55) + نوربورنن → [کوپلیمر]
```

**روش 3: اتصال حلقه‌های آزیدوفسفازن به پلی‌استایرن فسفونه‌دار**

اخیراً روشی توسعه یافته است برای **پیوند حلقه‌های آزیدو-فسفازن (۳.۵۶)** به پلی‌استایرن فسفونه‌دار.

**واکنش ۴۰:**
```
مرحله 1: تهیه پلی‌استایرن فسفونه‌دار
[—CH₂—CH(Ph)—]ₙ → [فسفونه کردن]
    → [—CH₂—CH(Ph—P(O)(OEt)₂)—]ₙ

مرحله 2: اتصال حلقه آزیدوفسفازن
حلقه فسفازن با گروه‌های N₃ (3.56)
    +
پلی‌استایرن فسفونه‌دار
    → [واکنش کلیک]
    → پلی‌استایرن—متصل به—حلقه‌های فسفازن
```

### ۵.۶ کوپلیمرهای Cyclolinear

#### **مفهوم (ساختار ۳.۱۸):**
```
حلقه—لینکر—حلقه—لینکر—حلقه—لینکر—حلقه
  P₃N₃   آلی    P₃N₃   آلی    P₃N₃
```

حلقه‌های فسفازن را نیز می‌توان در **پلیمرهای آلی از طریق کوپلیمرهای cyclolinear (۳.۱۸) و cyclomatrix (۳.۱۹)** گنجانید.

**گونه‌های Cyclolinear** از طریق واکنش‌های تریمرهای یا تترامرهای حلقوی سیکلوتری‌فسفازن یا تترافسفازن که فقط حاوی **دو گروه جانبی عاملی** هستند قابل دسترسی هستند، در حالی که گروه‌های جانبی دیگر **واحدهای غیرواکنشی** هستند.

#### **دو روش موجود:**

**روش 1: جایگزینی ناقص + پلیمریزاسیون تراکمی**

**مرحله 1: تهیه حلقه با دو کلر باقیمانده**

یک تریمر یا تترامر حلقوی سیکلوفسفازن که حاوی چهار یا شش گروه جانبی آلی بی‌اثر و **دو اتم کلر non-geminal** است ابتدا با واکنش (NPCl₂)₃ یا (NPCl₂)₄ با **کمبود استوکیومتری** یک آلکوکسید، آریلوکسید یا آمین تهیه می‌شود.
```
(NPCl₂)₃ + 2 NaOR → (NPCl₂)(NPOR₂)₂
                     دو Cl باقیمانده (non-geminal)
```

**مرحله 2: پلیمریزاسیون تراکمی - واکنش ۴۱**
```
(NPCl₂)(NPOR₂)₂ + HO—R'—OH
    حلقه با 2 Cl    دی‌ال
                 ↓
حلقه—O—R'—O—حلقه—O—R'—O—حلقه
  + HCl

واکنش کامل:
    OR  OR                        OR  OR
     |  |                          |  |
  R-P—N—P-R    + HO—(CH₂)ₙ—OH → —P—N—P—O—(CH₂)ₙ—O—
     |      |                      |      |
     N      N                      N      N
      \    /                        \    /
       P                             P
      / \                           / \
    Cl   Cl                        OR  OR
```

**تغییرات متعدد:**
تغییرات زیادی بر این رویکرد گزارش شده است.

**مشکلات این روش:**

**مشکل 1: کنترل تعداد کلر**
اطمینان از اینکه فقط **دو اتم کلر** روی سیکلوفسفازن باقی می‌مانند

**مشکل 2: استوکیومتری دقیق**
تضمین اینکه **دو مونومر در نسبت دقیق 1:1** حضور دارند

**نتایج عدم رعایت:**
- سه یا بیشتر کلر → **cross-linking** (شبکه‌ای شدن)
- مونومری در کمی اضافه → **الیگومرهای وزن مولکولی پایین**

**مشکل 3: طول زنجیر**
زنجیرهای پلیمر آلی بلند با گروه‌های الکوکسید یا آمینو انتهایی ممکن است نیاز به **زمان‌های واکنش بسیار طولانی** یا شرایط سخت داشته باشند تا بر **ناچیز بودن آماری** غلبه کنند که همه واحدهای واکنشی یکدیگر را در مخلوط واکنش پیدا کنند.

**روش 2: رویکرد ADMET (جدیدتر و بهتر)**

**واکنش ۴۲:**
```
مرحله 1: سنتز حلقه با زنجیرهای الفینی انتهایی
(NPCl₂)₃ + 2 NaO—(CH₂)ₙ—CH═CH₂
    → (NP(O(CH₂)ₙCH═CH₂)₂)(NPCl₂)
      حلقه با دو زنجیر الکوکسی با انتهای اولفین

مرحله 2: پلیمریزاسیون ADMET
کاتالیزور ارگانوفلزی ADMET:

n (حلقه—O—(CH₂)ₙ—CH═CH₂)₂
    → [کاتالیزور Grubbs/Schrock]
    → حذف اتیلن (C₂H₄)

حلقه—O—(CH₂)ₙ—CH═CH—(CH₂)ₙ—O—حلقه—O—(CH₂)ₙ—CH═CH—
    پلیمر cyclolinear
```

**ADMET = Acyclic Diene Metathesis** (متاتز دی‌ان غیرحلقوی)

**مزایا:**
- طول زنجیرهای حاصل از این روش عموماً **بلندتر** از آنهایی است که با تکنیک دهیدروهالوژناسیون تولید می‌شوند
- نیاز به کنترل استوکیومتری دقیق ندارد
- مشکلات حلالیت کمتر

### ۵.۷ کوپلیمرهای Cyclomatrix

#### **مفهوم (ساختار ۳.۱۹):**

اگر یک مولکول پیش‌ماده سیکلوفسفازن **بیش از دو جایگاه واکنشی** (مثل اتم‌های کلر) داشته باشد، واکنش‌های پیوند با دی‌آمین‌ها یا دی‌آلکوکسیدها یک **ساختار سه‌بعدی** به جای یک زنجیر خطی تولید خواهد کرد.
```
حلقه———حلقه———حلقه
  |   \ /   |   \ /   |
لینکر  ×  لینکر  ×  لینکر
  |   / \   |   / \   |
حلقه———حلقه———حلقه
  |       |       |
```

چنین موادی به عنوان **پلیمرهای cyclomatrix** شناخته می‌شوند.

#### **جایگاه در علم مواد:**

این گونه‌ها یک **مرحله میانی** بین پلیمرهای خطی و سرامیک‌ها را تشکیل می‌دهند، و یک موقعیت **حیاتی در علم مواد** را اشغال می‌کنند.

**اصطلاح "Ultrastructure":**
اصطلاح "ultrastructure" دامنه وسیعی از مواد این نوع را پوشش می‌دهد.

#### **تفاوت با پلیمرهای خطی:**

تفاوت بین یک پلیمر خطی یا ماکروسیکلیک و یک ultrastructure یکی از **سختی و پایداری حرارتی** است که هر دو با افزایش **چگالی cross-linking** افزایش می‌یابند.

#### **سرامیک‌ها:**

بیشتر سرامیک‌ها مانند:
- سیلیکات‌ها
- نیترید سیلیسیوم
- کاربید سیلیسیوم

**مواد کاملاً معدنی** هستند که توسط **آرایش سه‌بعدی فشرده** پیوندهای کووالانسی به هم متصل شده‌اند.

#### **Ultrastructure های آلی:**

با این حال، ultrastructure های آلی نیز شناخته شده‌اند.

**مثال:**
رزین ملامین-فرمالدئید که در آن **حلقه‌های s-تریازین** توسط یک چارچوب سه‌بعدی cross-link های آلی به هم متصل شده‌اند:
```
      N═C—N
       |   ‖
       C   C—NH—CH₂—NH—
      / \
     N   N
         |
    —CH₂—NH—
```

#### **سیستم‌های Cyclomatrix فسفازن:**

بسیاری از سیستم‌های cyclomatrix مختلف گزارش شده‌اند که بر اساس **حلقه‌های سیکلوتری‌فسفازن** به جای حلقه‌های s-تریازین هستند.

**روش عمومی:**
در این مواد، یک نوکلئوفیل دوعاملی، مانند یک دی‌آمین یا یک دی‌ال آروماتیک یا دی‌سدیم دی‌ال، برای **پیوند دادن حلقه‌های سیکلوفسفازن به یکدیگر** استفاده می‌شود.

**مثال خاص - واکنش ۴۳:**

سیکلوفسفازن‌ها با گروه‌های جانبی p-آمینوفنوکسی به راحتی با واکنش با **انیدرید مالئیک** به ultrastructure ها تبدیل می‌شوند:

حلقه فسفازن با گروه‌های —O—C₆H₄—NH₂

+ انیدرید مالئیک

حلقه—O—C₆H₄—NH—CO—CH═CH—CO—NH—C₆H₄—O—حلقه
|

۵.۸ سرامیک‌های معدنی مشتق شده از فسفازن‌ها
مفهوم:

مواد سرامیک‌مانند که حاوی هیچ واحد پیوندی آلی نیستند را می‌توان با پیرولیز (تجزیه حرارتی) آمینوفسفازن‌های حلقوی یا پلیمری سنگین تهیه کرد.

مثال - واکنش ۴۴:

[NP(NHR)₂]ₙ --[پیرولیز، دمای بالا، بی‌هوازی]--> 
                 ↓
        P-N (نیترید فسفر مانند)
        + محصولات آلی فرار

```
**تحت شرایط مناسب:**
محصولات پیرولیزی که با **فسفر-نیترید** مطابقت دارند تشکیل می‌شوند.

#### **سرامیک‌های چند عنصری:**

پلی‌فسفازن‌هایی که هم گروه‌های جانبی آمینو و هم **گروه‌های جانبی بورازین** را دارند، پس از پیرولیز **سرامیک‌های فسفر-نیتروژن-بور** تولید می‌کنند.
```
[NP(NH₂)(borazine)]ₙ --[پیرولیز]--> P-N-B سرامیک
```

#### **مزایای رویکرد "Polymer-to-Ceramic":**

تبدیل یک **پلیمر قابل شکل‌دهی** به یک سرامیک مزایای احتمالی زیادی برای **سنتز کنترل‌شده و ساخت سرامیک‌های پیشرفته** دارد.

**دلایل:**
1. پلیمر را می‌توان به شکل‌های پیچیده **قالب‌گیری** کرد
2. پلیمر را می‌توان به صورت **فیلم یا فیبر** تهیه کرد
3. سپس با پیرولیز، شکل حفظ می‌شود اما به سرامیک تبدیل می‌شود

**این اصل به تفصیل در فصل ۹ بحث می‌شود.**

## 🔬 بخش ششم: سیستم‌های هیبریدی از طریق کامپوزیت‌ها

### ۶.۱ مفهوم کامپوزیت‌ها

تشکیل ماکرومولکول‌های هیبریدی از طریق **پیوند پلیمرهای آلی به پلی‌فسفازن‌ها** فقط یکی از چندین رویکرد مختلف برای تولید سیستم‌های هیبریدی است.

**رویکرد دیگر:** که بر **علم مواد بیش از شیمی** تکیه دارد، شامل **مخلوط کردن دو پلیمر در حالت جامد** است - در این مورد یک پلی‌فسفازن با یک پلیمر آلی.

### ۶.۲ دو حالت قابل تصور

#### **حالت 1: مخلوط‌های پلیمری (Polymer Blends/Alloys)**

در اولین مورد، دو پلیمر **به صورت صمیمانه** مخلوط می‌شوند:
- **مکانیکی** 
- **یا** با ترکیب محلول‌های دو پلیمر و سپس **حذف حلال** با تبخیر
```
پلیمر A + پلیمر B → [مخلوط مکانیکی یا محلولی]
                    → [تبخیر حلال]
                    → مخلوط/آلیاژ پلیمری
```

**اصطلاحات:**
اینها **polymer blends** یا **polymer alloys** نامیده می‌شوند.

#### **حالت 2: شبکه‌های پلیمری درهم‌تنیده (IPNs)**

در رویکرد دوم، دو پلیمر ترکیب می‌شوند و سپس یک یا هر دو **cross-link می‌شوند** تا از جدا شدن زنجیرهای فردی و ایجاد جداسازی فازی **جلوگیری شود**.

**روش عملی:**
در عمل، این روش دوم با **متورم کردن نسخه cross-linked شده** پلیمر اول (یعنی پلی‌فسفازن) با **مونومر** دومی، و سپس **پلیمریزاسیون** و احتمالاً cross-linking جزء دوم حاصل می‌شود.
```
مرحله 1: پلی‌فسفازن cross-linked
[پلی‌فسفازن]ₙ → [cross-linking]
                → شبکه پلی‌فسفازن

مرحله 2: متورم کردن با مونومر
شبکه پلی‌فسفازن + مونومر آلی
    → متورم شدن

مرحله 3: پلیمریزاسیون درون شبکه
مونومر در شبکه → [پلیمریزاسیون]
                  → IPN
```

**اصطلاح:**
چنین سیستم‌های هیبریدی **interpenetrating polymer networks** یا **IPNs** نامیده می‌شوند.
```
تصویر IPN:
~~~~~شبکه A~~~~~
  ///شبکه B///
~~~~~شبکه A~~~~~
  ///شبکه B///
```

### ۶.۳ مثال‌های مخلوط‌های پلیمری فسفازن

**مثال‌های شناخته شده** از مخلوط‌های پلیمری فسفازن عبارتند از مواردی که در آن فسفازن‌ها با گروه‌های جانبی:
- متیل‌آمینو
- تری‌فلوئورواتوکسی
- فنوکسی
- الیگو-اتیلن‌اکسی

**مخلوط تشکیل می‌دهند با:**
- پلی(وینیل کلراید) - PVC
- پلی‌استایرن - PS
- پلی(متیل متاکریلات) - PMMA
- پلی(اتیلن اکساید) - PEO

**مثال‌های خاص:**
```
1. [NP(NHCH₃)₂]ₙ + PVC
2. [NP(OCH₂CF₃)₂]ₙ + PS
3. [NP(OPh)₂]ₙ + PMMA
4. [NP(OCH₂CH₂OCH₂CH₂OCH₃)₂]ₙ + PEO
```

### ۶.۴ مثال‌های IPN های فسفازن

**IPNها تولید شده‌اند از:**
- **MEEP**: `[NP(OCH₂CH₂OCH₂CH₂OCH₃)₂]ₙ`
- **و** پلی(متیل متاکریلات)
```
مرحله 1:
MEEP → [cross-link با UV یا گاما]
     → شبکه MEEP

مرحله 2:
شبکه MEEP + متیل متاکریلات (مونومر)
    → متورم شدن

مرحله 3:
→ [پلیمریزاسیون MMA در شبکه]
→ IPN MEEP/PMMA
```

### ۶.۵ نوع ویژه IPN: کامپوزیت‌های پلی‌فسفازن/سرامیک

یک نوع ویژه از IPN گزارش شده است که در آن یک **پلی‌فسفازن محلول در آب** مانند MEEP یک IPN با یک **شبکه سیلیکات یا تیتانات** تشکیل می‌دهد که از **هیدرولیز** تتراتوکسی‌سیلان یا تترااالکوکسی‌تیتانان تولید شده است.

**روش Sol-Gel:**
```
مرحله 1: تهیه محلول MEEP
MEEP در آب یا الکل

مرحله 2: افزودن پیش‌ماده سرامیک
MEEP محلول + Si(OEt)₄ یا Ti(OR)₄

مرحله 3: هیدرولیز و تراکم
Si(OEt)₄ + H₂O → SiO₂ شبکه + EtOH
           ↓
    MEEP به دام افتاده در شبکه سیلیکات
    → IPN پلیمر/سرامیک
```

**ساختار نهایی:**
```
—Si—O—Si—O—Si—
  |     |     |
  O MEEP O MEEP O
  |     |     |
—Si—O—Si—O—Si—
```

**خواص و کاربرد:**
این مواد به عنوان **مواد مناسب برای تهیه لایه‌های ضد الکتریسیته استاتیک** در تولید فیلم عکاسی توصیف شده‌اند.

**دلیل:**
- شبکه سیلیکات: مقاومت مکانیکی و پایداری
- MEEP: رسانایی یونی (جلوگیری از تجمع بار استاتیک)

## 🔗 بخش هفتم: پلی‌فسفازن‌های ارگانوفلزی

### ۷.۱ مشاهدات کلی

#### **جایگاه در علم مواد (شکل ۳.۶):**

پلیمرها یک بخش از **چهار حوزه اصلی علم مواد** را تشکیل می‌دهند:
```
        پلیمرها
           |
    ---------------
    |      |      |
  فلزات  سرامیک‌ها  کامپوزیت‌ها
```

**یکی از انگیزه‌های اصلی** برای کار اکتشافی تحقیقاتی با پلیمرهای معدنی، **توسعه حوزه‌ای است که بین پلیمرهای آلی، فلزات و سرامیک‌ها قرار دارد**.

**توسعه تحقیقات سنتز پلی‌فسفازن** با در نظر گرفتن این موضوع انجام شده است.

**مثال:**
استفاده از انواع وسیعی از **گروه‌های جانبی آلی** متصل به یک زنجیره پلی‌فسفازن اجازه داده است که موادی با خواص "organic-like" (شبیه آلی) توسعه یابند.

**در این بخش:**
پلی‌فسفازن‌هایی را در نظر می‌گیریم که **اتم‌های فلز در گروه جانبی** دارند و **رابط بین پلیمرها و فلزات** را اشغال می‌کنند.

### ۷.۲ سه رویکرد برای سنتز متالوفسفازن‌ها

**سه رویکرد** برای سنتز متالوفسفازن‌ها بررسی شده‌اند که با سه واحد تکراری نشان داده شده به عنوان **۳.۵۷-۳.۶۱** خلاصه می‌شوند:

#### **رویکرد 1: ترکیبات کوئوردیناسیون (۳.۵۷، ۳.۵۸)**
```
        R    R
        |    |
    —N═P—N═P—
        ↑    ↑
        M    M
        
فلز به نیتروژن اسکلت هماهنگی می‌کند
```

#### **رویکرد 2: فلزات متصل از طریق گروه‌های فاصله‌انداز (۳.۵۹-۳.۶۱)**
```
        R         R
        |         |
    —N═P——N═P—
        |         |
      Spacer    Spacer
        |         |
        M         M
```

**انواع spacer:**
- **۳.۵۹:** P-P bond (پیوند فسفر-فسفر)
- **۳.۶۰:** آریل-فسفین
- **۳.۶۱:** تیواتر یا سایر پیوندکننده‌ها

#### **رویکرد 3: پیوند مستقیم فلز-فسفر (۳.۶۷، ۳.۶۸)**
```
        R    M    R    M
        |    |    |    |
    —N═P—N═P—N═P—N═P—
        |         |
        R         R
```

### ۷.۳ ترکیبات کوئوردیناسیون

#### **الف) هماهنگی پلاتین به زنجیره اصلی**

**ساده‌ترین رویکرد مفهومی:**
آنی که در **۳.۵۷** نشان داده شده، که در آن از **الکترون‌های جفت‌نشده نیتروژن‌های اسکلتی** برای هماهنگی اتم‌های فلز به اسکلت استفاده می‌شود.
```
        R         R
        |         |
    —N═P——N═P—
     ↑↑        ↑↑
    جفت الکترون‌های آزاد
```

**فقط یک مثال** از این آرایش داده خواهد شد، اگرچه این اصل می‌تواند به طور گسترده اعمال شود.

**مثال: عامل ضد تومور پلاتین (۳.۵۸)**

**پلیمر پایه:** پلی‌فسفازن محلول در آب نشان داده شده در **۳.۳۲**:
```
[NP(NHR)₂]ₙ
R = آلکیل
```

**واکنش:**
```
[NP(NHCH₂CH₂NH₂)₂]ₙ + PtCl₂/18-crown-6 (آبی)
                        ↓
        [NP(NHCH₂CH₂NH₂)₂]ₙ—PtCl₂  (3.58)
         عامل ضد تومور پلیمری تجربی
```

**محدودیت بارگذاری:**
فقط **بارگذاری کم** PtCl₂ قابل تحمل است (تقریباً یک اتم پلاتین به ازای هر ۱۷ واحد تکراری) چون مقادیر بیشتر تمایل به **cross-link کردن زنجیرها** دارند و سیستم را نامحلول می‌کنند.

**جایگاه هماهنگی:**
شواهد از مطالعات ترکیب مدل نشان می‌دهد که احتمالاً **اتم‌های نیتروژن اسکلتی** به جای اتم‌های نیتروژن گروه جانبی **جایگاه‌های هماهنگی** را تشکیل می‌دهند.

**دلیل:**
این احتمالاً نتیجه **عرضه قوی الکترون از گروه‌های جانبی آمینو** است که **خصلت هماهنگی اتم‌های نیتروژن اسکلتی** را افزایش می‌دهد.
```
        NHCH₂CH₂NH₂
         |
    —N═P—  ← الکترون‌ها از NH₂ به P
     ↑      ← چگالی الکترونی N اسکلتی ↑
    PtCl₂ هماهنگی بهتر
```

#### **ب) فلزات متصل به پلیمر از طریق واحدهای فاصله‌انداز آلی یا معدنی**

تعدادی از **گروه‌های جانبی آلی** به عنوان واحدهای "spacer" برای پیوند فلزات واسطه به یک زنجیر پلی‌فسفازن استفاده شده‌اند.

**برخی از واحدهای spacer** در ساختارهای **۳.۵۹-۳.۶۱** نشان داده شده‌اند.

**نوع 1: پیوند P-P (۳.۵۹)**
```
        PPh₂      PPh₂
         |         |
    —N═P——N═P—
         |         |
        OCH₂CF₃   OCH₂CF₃
         ↓         ↓
        M         M
```

گونه‌های نوع **۳.۵۹** که در آن یک واحد دهنده فسفین با **پیوند P-P** به یک اتم فسفر اسکلتی متصل است، **کمترین گونه‌های پایدار** و **دشوارترین برای تهیه** هستند.

**روش سنتز:**
به طور خاص، آنها با واکنش پیوندهای P-Cl در پلیمر با **لیتیوم تری‌اتیل‌بوروهیدرید (LiBEt₃H)** سنتز می‌شوند که یک جایگاه P⁻ به جای پیوند P-Cl تولید می‌کند.
```
مرحله 1:
[NP(Cl)(OCH₂CF₃)]ₙ + LiBEt₃H 
    → [NP(H)(OCH₂CF₃)]ₙ
      → [NP⁻Li⁺(OCH₂CF₃)]ₙ

مرحله 2:
[NP⁻(OCH₂CF₃)]ₙ + Ph₂PCl 
    → [NP(PPh₂)(OCH₂CF₃)]ₙ  (3.59 بدون فلز)

مرحله 3:
[NP(PPh₂)(OCH₂CF₃)]ₙ + Ru₃(CO)₁₂
    → [NP(PPh₂-Ru₃(CO)₁₁)(OCH₂CF₃)]ₙ
```

**نوع 2: واحدهای فاصله‌انداز آریل-فسفین (۳.۶۰، ۳.۶۲)**

واحدهای spacer در **۳.۶۰** از پلی‌فسفازن‌هایی که دارای گروه‌های جانبی **p-برموفنوکسی** هستند از طریق یک واکنش **لیتیایی** و تیمار با یک **دی‌ارگانوکلروفسفین** ساخته می‌شوند تا **۳.۶۲** را بدهند.

**شیمی (دنباله واکنش ۴۵):**
```
مرحله 1: لیتیایی کردن
[NP(O-C₆H₄-Br-p)₂]ₙ + n-BuLi
    → [NP(O-C₆H₄-Li-p)₂]ₙ

مرحله 2: افزودن فسفین
[NP(O-C₆H₄-Li-p)₂]ₙ + R₂PCl

[NP(O-C₆H₄-PR₂-p)₂]ₙ + کمپلکس فلزی

```

**پلیمر ۳.۶۲** به انواع مختلفی از گونه‌های متالو هماهنگی می‌کند، از جمله:
- **ترکیبات کلاستر اسمیوم**
- **کاتالیزورهای هیدروفرمیلاسیون کبالت کربونیل**

**کاربرد به عنوان کاتالیزور:**
وقتی به عنوان یک **کاتالیزور هیدروفرمیلاسیون پلیمری** استفاده شد، این گونه اخیر ثابت کرد که زنجیره پلی‌فسفازن تحت **شرایط سخت** که اغلب برای این نوع واکنش‌ها مورد نیاز است چقدر **پایدار** است.

**ضعیف‌ترین پیوندها:**
ضعیف‌ترین پیوندها در مولکول ثابت شد که آنهایی بین **اتم‌های فسفر فسفین** و **گروه‌های فاصله‌انداز آروماتیک** هستند.

**نوع 3: زنجیرهای جانبی تیواتر**
```

```
**نوع 4: واحدهای جانبی فروسنیل و روتنوسنیل (۳.۶۳، ۳.۶۴)**

پیوند واحدهای جانبی فروسنیل و روتنوسنیل به یک پلی‌فسفازن از طریق شیمی نشان داده شده در **واکنش‌های (۴۶) و (۴۷)** انجام شده است.

**واکنش ۴۶:**
```
(NPCl₂)₃ + فروسن—لیتیایی‌شده

→ [NP(Cl)(فروسنیل)]ₙ

→ [+ NaOCH₂CF₃]

```
**واکنش ۴۷:**
```

```
**۳.۶۳:** متالوسن از طریق **یک پیوند کربن-فسفر** به پلیمر متصل است
```
        OCH₂CF₃   Fc
         |        |
    —N═P——N═P—
         |        |
        OCH₂CF₃  OCH₂CF₃

Fc = فروسنیل (Fe(C₅H₅)(C₅H₄)—)
```

**۳.۶۴:** متالوسن **پلی** بین دو اتم فسفر مجاور تشکیل می‌دهد
```
        OCH₂CF₃
         |
    —N═P——N
         |   \
        Fe    P—
       /  \   |
      C₅H₄ C₅H₄ OCH₂CF₃
```

**اهمیت پلیمرهای ۳.۶۳ و ۳.۶۴:**

**دلیل 1: دوپ اکسیداتیو**
آنها می‌توانند به صورت اکسیداتیو با ید **دوپ** شوند تا پلیمرهایی تولید کنند که **نیمه‌رساناهای ضعیف** هستند.
```
[پلیمر-Fc]ₙ + I₂ → [پلیمر-Fc⁺]ₙ I⁻
                    نیمه‌رسانا
```

**دلیل 2: پوشش‌های واسطه الکترود**
وقتی روی سطح یک الکترود رسوب داده شوند، پلیمرها به عنوان **پوشش‌های "واسطه (mediator)" الکترود** عمل می‌کنند که **انتقال الکترون** بین یک واکنش‌دهنده در محلول و الکترود را **کمک می‌کنند**.
```
محلول → فروسن در پلیمر → الکترود
        (واسطه انتقال e⁻)
```

**چرا؟**
فروسن آزاد، به عنوان مثال، در **محیط الکترود سطحی باقی نمی‌ماند**. یک **پلیمر غیرفعال الکتروشیمیایی** برای **تثبیت متالوسن** در ناحیه سطح الکترود مورد نیاز است.

**کوپلیمریزاسیون:**
کوپلیمریزاسیون تریمرهای فسفازن که دارای گروه‌های جانبی فروسنیل و روتنوسنیل هستند امکان‌پذیر است، و پلیمرهای حاصل دارای **جایگاه‌هایی با پتانسیل‌های اکسیداسیون متفاوت** هستند.
```
[NP(فروسن)(OCH₂CF₃)]—[NP(روتنوسن)(OCH₂CF₃)]
     E₀ = 0.4V            E₀ = 0.8V

```
**خواص لاستیک معدنی:**
```

```
### **۱۹۶۰-۱۹۶۶: انقلاب Allcock** (صفحات ۱۴-۱۷)

**Allcock، Kugel و Valan** سه مقاله کلیدی منتشر کردند:

#### **تفکر ۱: درک ناپایداری**
```
ناپایداری هیدرولیتیکی = واکنش‌پذیری بالای P-Cl
                         ↓
              فرصت برای جایگزینی!
              
P-Cl + گروه آلی → P-R (پایدار)
```

**تلاش اولیه:**
- جایگزینی Cl با گروه‌های آلکوکسی
- نتیجه: دلگرم‌کننده بود
- **مشکل**: جایگزینی کامل نبود (چون حل نمی‌شد)

#### **تفکر ۲: حل مشکل حلالیت** (صفحه ۱۵)

**تحلیل:**
```
متورم شدن + عدم حلالیت = cross-linked
```

**راه حل:**
باید پلیمر را **بدون cross-link** بسازیم!

**کشف کلیدی** (صفحه ۱۵، واکنش ۲):
```
(NPCl₂)₃ --[۲۵۰°C، کنترل دقیق]--> (NPCl₂)ₙ
  حلقه     - زمان                    خطی
           - دما                     (محلول!)
           - خلوص
           - توقف قبل از ۷۰٪

(NPCl₂)ₙ --[گرما بیشتر]--> cross-linked
  محلول                      نامحلول
```

**نکات کنترل:**
- ⚠️ کنترل دقیق **زمان**
- ⚠️ کنترل دقیق **دما** (۲۱۰-۲۵۰°C)
- ⚠️ **خلوص** تریمر حلقوی
- ⚠️ **توقف قبل از ~۷۰٪** تبدیل به پلیمر

**نتیجه:**
- پلیمر خطی (۳.۲۱) که **کاملاً محلول** است در:
  - بنزن
  - تولوئن
  - تتراهیدروفوران (THF)

#### **تفکر ۳: جایگزینی ماکرومولکولی** (صفحه ۱۶، واکنش ۳)
```
(NPCl₂)ₙ + نوکلئوفیل آلی → پلیمر جایگزین‌شده
  محلول                      + NaCl↓
```

**نوکلئوفیل‌ها:**
- نمک‌های سدیمی الکل‌ها: NaOR
- نمک‌های سدیمی فنل‌ها: NaOAr  
- آمین‌های اولیه/ثانویه: RNH₂, R₂NH

**شگفتی:**
به طور متوسط **۳۰,۰۰۰ اتم کلر** در هر مولکول جایگزین می‌شوند!

#### **تفکر ۴: پایداری پلیمرهای مشتق**

**نتایج:**
```

```

### **مشکل واکنش‌های ارگانوفلزی** (صفحات ۲۶-۳۳)

#### **دو مسیر رقیب** (صفحه ۲۶، واکنش ۱۰):
```

(NPCl₂)ₙ + RLi یا RMgX → ۱) جایگزینی Cl با R ✅
                         ۲) شکستن P-N اسکلت ❌
```

#### **مکانیسم شکستن زنجیر** (صفحه ۲۷-۲۸):

**عامل کلیدی:** coordination RLi به N اسکلتی
```

        R         R
        |         |

     ↑↑

     ↓
  شکستن پیوند P-N

```
**عوامل موثر:**

**۱. الکترون‌های جفت‌نشده N:**
- چگالی الکترونی بالا → coordination آسان → شکستن زیاد

**۲. نقش Cl:**
```
Cl = الکترون‌کش قوی
   ↓
کاهش چگالی e⁻ روی N
   ↓
محافظت از اسکلت
```

**۳. خطر پس از جایگزینی:**
```

   ↓

   ↓


      / \                       / \

```
**مرحله ۲: باز شدن حلقه و انتشار:**
```

```
- ❌ **گران‌ترین** روش
```

```
## ۳.۱ واکنش‌های سطحی (صفحات ۶۸-۷۲)

### **مشکل عمومی** (صفحه ۶۸):

**سناریوی معمول:**
```
پلیمری با خواص حجمی عالی
    +
خواص سطحی نامناسب
    =
نمی‌توان استفاده کرد!
```

**مثال‌ها:**
1. کشسانی عالی **اما** سطح باعث لخته شدن خون می‌شود
2. سازگاری زیستی سطحی عالی **اما** خیلی شکننده

**راه حل:**
```
انتخاب پلیمر برای خواص حجمی
    +
اصلاح سطح بدون تأثیر بر حجم
    =
پلیمر ایده‌آل!
```

### **چرا پلی‌فسفازن مناسب است؟** (صفحه ۶۹)

**۱. تنوع گروه‌های جانبی اولیه**
**۲. قابلیت تبدیل آلی گسترده**
**۳. محافظت اسکلت توسط گروه‌های جانبی**

**کاربردها:**
- پزشکی
- پوشش‌های سطحی
- چسبندگی
- مقاومت در برابر آب و هوا
- تثبیت آنزیم‌ها

### **واکنش‌های سطحی نمونه** (صفحات ۷۰-۷۲، واکنش‌های ۲۹-۳۳):

#### **واکنش ۲۹: تبدیل OCH₂CF₃ → OH:**
```
سطح: —N═P(OCH₂CF₃)₂— + NaOH/Bu₄NBr (آبی)
           ↓
      —N═P(OH)₂—

آبگریز (زاویه ≈100°) → آبدوست (زاویه ≈0°)
```

**کاربرد:**
تبدیل سطح غیرچسبنده به چسبنده

#### **واکنش ۳۰: افزودن آب به اولفین:**
```
—N═P—CH₂—CH═CH₂ + H₂O → —N═P—CH₂—CH(OH)—CH₃
```

#### **واکنش ۳۱: نیتراسیون/کاهش سطحی:**
```
[NP(OPh)₂]ₙ سطح + HNO₃ → [NP(OC₆H₄NO₂)₂]ₙ سطح
                          ↓ [H]
                    [NP(OC₆H₄NH₂)₂]ₙ سطح
```

**کاربرد:** گروه‌های NH₂ سطحی برای اتصال مولکول‌های زیستی

#### **واکنش ۳۲: سولفوناسیون سطحی:**
```
[NP(OPh)₂]ₙ سطح + H₂SO₄ (غلیظ) → [NP(OC₆H₄SO₃H)₂]ₙ سطح

زاویه تماس: 101° → 0°
آبگریز → آبدوست (در یک مرحله!)
```

**⚠️ نکته مهم:**
کنترل دقیق شرایط برای جلوگیری از:
- انحلال پلیمر در اسید قوی
- شکستن زنجیر

#### **واکنش ۳۳: تثبیت هپارین:**
```
سطح—NH₂ + هپارین → سطح—NH—هپارین

نتیجه: افزایش 5 برابری زمان انعقاد خون
```

**مثال دیگر: تثبیت دوپامین:**
```
سطح + دوپامین → سطح—دوپامین
نتیجه: همان پاسخ سلولی مثل دوپامین آزاد
```

### **نکته مهم: حرکات مولکولی** (صفحه ۷۲):

**مسئله احتمالی:**
```
سطح آبدوست روی پلیمر آبگریز:

در هوای خشک: واحدهای آبدوست دفن می‌شوند
در مایع آبدوست: واحدها به سطح بر می‌گردند
```

**فرآیند برگشت‌پذیر است!**

### **۳.۲.۱ مفهوم و روش‌ها** (صفحه ۷۳-۷۴)

**هدف:**
ترکیب خواص ویژه پلی‌فسفازن‌ها با:
- پلیمرهای آلی کلاسیک
- پلی(ارگانوسیلوکسان)ها

**مزایای احتمالی:**
- کاهش هزینه

  - مقاومت در برابر آتش
  - آب‌گریزی
  - سهولت اتصال عوامل زیستی

**روش‌های هیبریدسازی:**

**۱. کوپلیمرهای Block:**
- استفاده از living cationic polymerization
- تبدیل به ماکرومولکول‌های عامل‌دار انتهایی

**۲. کوپلیمرهای Comb/Graft:**
- پلی‌فسفازن با گروه‌های vinyl/allyl/norbornenyl انتهایی

**۳. پلیمرهای آلی با حلقه‌های فسفازن جانبی:**
- عامل‌دار کردن حلقه‌های کوچک با vinyl/allyl/norbornenyl
- پلیمریزاسیون

**۴. مواد Cyclolinear/Cyclomatrix:**
- اتصال حلقه‌های فسفازن با پیوندهای آلی

### **۳.۲.۲ کوپلیمرهای Block** (صفحات ۷۵-۸۳)

#### **اصل** (صفحه ۷۶):

**Living polymerization** مولکول‌های زنده تولید می‌کند که با **خاتمه‌دهنده‌های فسفورانیمین** واکنش می‌دهند

**خاتمه‌دهنده‌ها/آغازگرها:**
```
فرمول کلی: Me₃SiN=PR₂R'

R = گروه غیرواکنشی (مثل OCH₲CF₃)
R' = واحد عاملی
```

**مثال‌های ساختار** (صفحه ۷۷، ۳.۴۴-۳.۴۹):
```
3.44: Me₃SiN=P(OCH₂CF₃)₂—CH₂—CH═CH₂ (آلیل)
3.45: Me₃SiN=P(OCH₂CF₃)₂—Ph—CH═CH₂ (استایرنیل)
3.46: Me₃SiN=P(OCH₂CF₃)₂—(CH₂)ₙ—CH═CH₂ (وینیل)
3.47: Me₃SiN=P(OCH₂CF₃)₂—نوربورنن
3.48: Me₃SiN=P(OCH₂CF₃)₂—(CH₂)ₙ—OH
3.49: Me₃SiN=P(OCH₂CF₃)₂—(CH₂)ₙ—COOH
```

**شرط مهم:**
R' نباید هالوژن X باشد (که به صورت Me₃SiX حذف می‌شود)

#### **توالی واکنش** (صفحه ۷۸، واکنش ۳۴):
```
مرحله 1: پلیمریزاسیون
PCl₅ + Me₃SiN=PCl₃ → [—N=PCl₂—]ₙ⁺
                      زنجیر زنده

مرحله 2: خاتمه
[—N=PCl₂—]ₙ⁺ + Me₃SiN=P(OCH₂CF₃)₂—R'
    ↓
[—N=PCl₂—]ₙ—N=P(OCH₂CF₃)₂—R'
 پلیمر با گروه عاملی انتهایی

مرحله 3: جایگزینی Cl ها
+ NaOR" → [—N=P(OR")₂—]ₙ—N=P(OCH₂CF₃)₂—R'
```

**⚠️ نکته خیلی مهم** (صفحه ۷۹):
پس از خاتمه، Cl های زنجیر **باید** جایگزین شوند قبل از مرحله پیوند انتهایی

#### **انواع ساختار** (صفحه ۷۹):

**بسته به نسبت PCl₅/مونومر:**

**یک انتهای زنده:**
```
P—[پلیمر]⁺ → dیبلوک
```

**دو انتهای زنده:**
```
⁺[پلیمر]⁺ → تری‌بلوک
```

**ساختارهای عمومی:**
```
3.12: [فسفازن]—Sw—[آلی] (دی‌بلوک)
3.13: [آلی]—Sw—[فسفازن]—Sw—[آلی] (A-B-A)
3.14: [فسفازن]—Sw—[آلی]—Sw—[فسفازن] (B-A-B)

Sw = switchover group
```

#### **روش جایگزین** (صفحات ۸۰-۸۱، واکنش ۳۵):
```
مرحله 1: اتصال آغازگر به پلیمر آلی
پلیمر آلی—OH + Cl—Si(CH₳)₂—N=P(OCH₂CF₃)₂Cl
    ↓
پلیمر آلی—O—Si(CH₃)₂—N=P(OCH₂CF₃)₂Cl
         ماکروآغازگر

مرحله 2: رشد زنجیر فسفازن
+ Me₃SiN=PCl₃ + PCl₅
    ↓
پلیمر آلی—Sw—[پلی‌فسفازن]
```

#### **نمونه‌های سنتز شده** (صفحه ۸۲، ۳.۵۰-۳.۵۳):
```
3.50: [فسفازن]—Sw—[پلی(اتیلن اکساید)]
3.51: [فسفازن]—Sw—[پلی‌استایرن]
3.52: [PDMS]—Sw—[فسفازن]—Sw—[PDMS]
3.53: [پلی‌متاکریلات]—Sw—[فسفازن]

خواص و کاربردها (صفحه ۸۳):

Block copolymers با بلوک‌های مختلف (مثل آبگریز/آبدوست):

می‌توانند:

    سیستم‌های جامد جدا شده فازی تشکیل دهند
    میسل‌ها یا وزیکول‌ها در محیط آبی تشکیل دهند

فرصت طراحی: با کنترل طول هر بلوک → تغییرات خاصیت نامحدود

#### **روش‌های سنتز:**

**۲. پلیمریزاسیون گروه‌های عاملی:**

#### **مثال‌ها** (صفحات ۸۵-۸۶):

```
[—N=P(OCH₂CF₳)₂—]ₙ—N=P(OCH₂CF₃)₂—Ph—CH═CH₂
```

```
                                    → اتصال
                                    → کوپلیمر graft
```

### **۳.۲.۴ پلیمرهای آلی با حلقه‌های فسفازن** (صفحات ۸۷-۹۰)

#### **مفهوم و اهمیت** (صفحه ۸۷):

**حلقه تریمری فسفازن = واحد پایدار با خواص ارزشمند:**
- ✅ مقاومت در برابر گرما
- ✅ رفتار بازدارنده شعله

**نتیجه:**
```
پلیمر آلی + حلقه فسفازن = قابلیت اشتعال کمتر
```

**Rayon مقاوم در برابر آتش:**
- سلولز + آلکوکسی‌سیکلوفسفازن‌ها (به دام افتاده فیزیکی)

**برای کاربردهای بلندمدت:**
اتصال **کووالانسی** حلقه‌ها به زنجیر بهتر است (فرار نمی‌کند)

#### **سه روش توسعه یافته:**

**روش ۱: حلقه با یک گروه vinyl/allyl** (صفحه ۸۸، واکنش ۳۸):

**مونومر ۳.۵۴:**
```
حلقه فسفازن—O—(CH₂)ₙ—CH═CH₂

پلیمریزاسیون:
n (مونومر) → —CH₂—CH—CH₂—CH—
                   |        |
                 حلقه    حلقه
```

**روش ۲: گروه‌های norbornenyl** (صفحه ۸۹، واکنش ۳۹):

**مونومر ۳.۵۵:**
```
حلقه فسفازن—O—(CH₲)ₙ—نوربورنن

ROMP:
n (مونومر) + کاتالیزور Grubbs
    → [پلی‌نوربورنن با حلقه‌های فسفازن]

یا کوپلیمریزاسیون:
+ نوربورنن → کوپلیمر
```

**روش ۳: اتصال حلقه‌های آزیدو** (صفحه ۹۰، واکنش ۴۰):
```
پلی‌استایرن فسفونه‌دار + حلقه آزیدوفسفازن (3.56)
    --[واکنش کلیک]-->
    پلی‌استایرن—حلقه‌های فسفازن
```

### **۳.۲.۵ کوپلیمرهای Cyclolinear** (صفحات ۹۱-۹۵)

#### **مفهوم** (صفحه ۹۱، ساختار ۳.۱۸):
```
حلقه—لینکر—حلقه—لینکر—حلقه—لینکر—حلقه
  P₃N₃   آلی    P₃N₃   آلی    P₃N₃
```

**دسترسی از طریق:**
واکنش تریمرها/تترامرهای حلقوی با **فقط دو گروه جانبی عاملی**

#### **دو روش:**

**روش ۱: جایگزینی ناقص + تراکم** (صفحات ۹۲-۹۴، واکنش ۴۱):
```
مرحله 1: تهیه حلقه با 2 Cl باقیمانده
(NPCl₂)₃ + 2 NaOR → (NPCl₂)(NPOR₂)₂
                     دو Cl non-geminal

مرحله 2: پلیمریزاسیون تراکمی
(NPCl₂)(NPOR₂)₂ + HO—R'—OH
    ↓
حلقه—O—R'—O—حلقه—O—R'—O—حلقه + HCl
```

**مشکلات** (صفحه ۹۴):

**۱. کنترل تعداد Cl:**
- اطمینان از فقط **2 کلر** روی حلقه
- 3+ کلر → cross-linking
- کمتر از 2 → مشکل

**۲. استوکیومتری:**
- نسبت دقیق 1:1 مونومرها
- کمی اضافه → الیگومرهای کوتاه

**۳. طول زنجیر:**
- زنجیرهای بلند آلی → زمان واکنش طولانی
- مشکل آماری یافتن یکدیگر

**روش ۲: رویکرد ADMET (بهتر)** (صفحه ۹۵، واکنش ۴۲):
```
مرحله 1: سنتز حلقه با زنجیرهای الفینی انتهایی
حلقه—O—(CH₂)ₙ—CH═CH₂ (دو طرف)

مرحله 2: ADMET
کاتالیزور Grubbs/Schrock
    ↓
حذف C₂H₄
    ↓
حلقه—O—(CH₂)ₙ—CH═CH—(CH₂)ₙ—O—حلقه—...
     پلیمر cyclolinear
```

**ADMET = Acyclic Diene Metathesis**

**مزایا:**
- ✅ طول زنجیر بلندتر
- ✅ کنترل استوکیومتری لازم نیست
- ✅ مشکلات حلالیت کمتر

### **۳.۲.۶ کوپلیمرهای Cyclomatrix** (صفحات ۹۶-۹۹)

#### **مفهوم** (صفحات ۹۶-۹۷، ساختار ۳.۱۹):
```
حلقه———حلقه———حلقه
  |   \ /   |   \ /   |
لینکر  ×  لینکر  ×  لینکر
  |   / \   |   / \   |
حلقه———حلقه———حلقه
  |       |       |
```

**اگر حلقه پیش‌ماده >2 جایگاه واکنشی داشته باشد:**
→ ساختار سه‌بعدی (نه خطی)
→ Cyclomatrix polymer

#### **جایگاه در علم مواد** (صفحه ۹۷):
```
پلیمرهای خطی ← Ultrastructure → سرامیک‌ها
```

**تفاوت:**
- سختی
- پایداری حرارتی  
- هر دو با افزایش **چگالی cross-linking** افزایش می‌یابند

#### **سرامیک‌ها** (صفحه ۹۸):

**مثال‌های کاملاً معدنی:**
- سیلیکات‌ها
- نیترید سیلیسیوم (Si₃N₄)
- کاربید سیلیسیوم (SiC)

**نگهداری توسط:** آرایش ۳D فشرده پیوندهای کووالانسی

**Ultrastructure های آلی:**
**مثال:** رزین ملامین-فرمالدئید
- حلقه‌های s-تریازین
- متصل با چارچوب ۳D cross-link های آلی

#### **سیستم‌های Cyclomatrix فسفازن** (صفحه ۹۹، واکنش ۴۳):

**روش عمومی:**
نوکلئوفیل دوعاملی (دی‌آمین/دی‌ال) برای پیوند حلقه‌ها

**مثال:**
```
حلقه با گروه‌های p-آمینوفنوکسی + انیدرید مالئیک
    ↓
حلقه—O—C₆H₄—NH—CO—CH═CH—CO—NH—C₆H₄—O—حلقه
                 |
            cross-links ثلاثی
```

**خواص:**
- پایداری حرارتی بسیار بالا
- مناسب برای پوشش‌های دمای بالا

### **۳.۲.۷ سرامیک‌های معدنی** (صفحه ۱۰۰، واکنش ۴۴):

**روش:**
پیرولیز (تجزیه حرارتی) آمینوفسفازن‌ها
```
[NP(NHR)₂]ₙ --[پیرولیز، بی‌هوازی، دمای بالا]-->
                 ↓
        P-N (نیترید فسفر) + محصولات آلی فرار
```

**سرامیک‌های چند عنصری:**
```
[NP(NH₂)(بورازین)]ₙ --[پیرولیز]--> P-N-B سرامیک
```

**مزایای "Polymer-to-Ceramic":**
1. پلیمر قابل شکل‌دهی
2. قالب‌گیری به شکل‌های پیچیده
3. تهیه فیلم/فیبر
4. پیرولیز → حفظ شکل + تبدیل به سرامیک

## ۵.۴ کانفورماسیون ماکرومولکولی (صفحات ۱۳۸-۱۴۲)

### **تعریف** (صفحه ۱۳۸):

**کانفورماسیون زنجیر تعیین می‌شود توسط:**
زوایای پیچشی (torsional angles) پیوندهای اسکلتی

### **قرارداد** (صفحه ۱۳۸-۱۳۹، شکل ۳.۹):

**زوایای 0°, 0°:**
```
trans-trans-planar conformation

        R    R    R    R
        |    |    |    |
    —N—P—N—P—N—P—N—P—
```

**پیچش پیوندهای 2 و 4 به 180°:**
```
cis-trans-planar conformation

    R         R         R
    |         |         |
—N—P———N—P———N—P—
    |         |         |
    R         R         R
```

**زوایای دیگر:**
- مارپیچ‌های spiral
- کویل تصادفی (اگر زوایا منظم نباشند)

### **عوامل تعیین‌کننده** (صفحه ۱۴۰):

**۱. دفع/جذب بین R های نزدیک روی همان زنجیر**
**۲. نیروهای بسته‌بندی زنجیرها در حوزه میکروکریستالین**
**۳. مانع cis-trans (اگر پیوند دوگانه pπ-pπ وجود داشته باشد)**

### **ساده‌سازی در فسفازن‌ها** (صفحات ۱۴۱-۱۴۲):

**تفاوت با پلیمرهای آلی:**

**۱. موقعیت R:**
```
فسفازن: R به یک در میان اتم‌های اسکلتی
آلی معمولی: R به همه اتم‌های اسکلتی
```

**۲. نوع پیوند:**
```
فسفازن: نوشته می‌شود —N═P—
        اما پیوند از نوع کلاسیک pπ-pπ نیست
        مانع پیچشی اسکلتی وجود ندارد!

همه پیوندهای اسکلت مثل single bond رفتار می‌کنند
(حداقل از نظر خصوصیات پیچشی)
```

**نتیجه:**
```
کانفورماسیون فسفازن قابل فهم است عمدتاً از طریق:
دفع/جذب بین R های متصل به P های نزدیک
```

### **کانفورماسیون ترجیحی:**

**cis-trans-planar** (شکل ۳.۹):
```
این کانفورماسیون:
R ها حداکثر فاصله از یکدیگر
    ↓
کمینه کردن دفع‌های داخلی
    ↓
کمترین انرژی
```

**تایید:**
- محاسبات مکانیک مولکولی
- داده‌های XRD (فاصله تکرار ≈ 4.9 Å)

## ۵.۵ پیوندها در پلی‌فسفازن‌ها (صفحات ۱۴۳-۱۴۵)

### **تفاوت با پلیمرهای آلی** (صفحه ۱۴۳):

**الکترون‌های ظرفیت در یک قطعه:**
```
هر P: 5 الکترون ظرفیت
هر N: 5 الکترون ظرفیت

اگر 2 e⁻ از N در orbital جفت‌نشده باشد
و جفت‌های e⁻ به چارچوب σ اختصاص یابد:
    ↓
2 الکترون باقی می‌ماند (1 از P، 1 از N)
```

### **مدل پیوندی** (صفحه ۱۴۴، ۳.۷۰-۳.۷۱):

**این الکترون‌ها جفت نشده نمی‌مانند:**
```
e⁻ روی N: در orbital 2pz
e⁻ روی P: در orbital 3d
    ↓
تشکیل پیوند π
```

**ویژگی‌های مهم:**
```
پیوندهای π روی "جزیره‌های" 3 اتمی
غیرمتمرکز می‌شوند

اما
روی کل زنجیر به طور گسترده غیرمتمرکز نمی‌شوند
```

**چرا؟**
```
عدم تطابق orbital + گره‌ها در هر P
```

**نتیجه مهم:**
```
P می‌تواند تا 5 orbital 3d استفاده کند
    ↓
پیچش پیوند P-N:
p-orbital N می‌تواند با d-orbital در
تقریباً هر زاویه پیچشی همپوشانی کند
    ↓
مانع پیچشی ذاتی بسیار کوچک‌تر از
پیوند دوگانه pπ-pπ آلی
```

**محاسبات** (صفحه ۱۴۵):
```
مانع پیچشی ذاتی در پیوندهای اسکلت
    ≈ 0.1 kcal/bond
    (بسیار کم!)
```

### **توضیح خواص:**

**ساختار "island" dπ-pπ ممکن است توضیح دهد که چرا:**
```
1. بیشتر پلی‌فسفازن‌ها بی‌رنگ هستند (نه رنگی)
2. عایق هستند (نه رسانای الکترونیکی)

استثناها:
- پلیمرها با کروموفورهای گروه جانبی
- گونه‌هایی با واحدهای جانبی الکتروفعال
```

## ۵.۶ انعطاف‌پذیری اسکلتی (صفحات ۱۴۶-۱۴۹)

### **ارتباط با Tg** (صفحه ۱۴۶):

**تعریف Tg (دمای گذار شیشه‌ای):**
```
زیر Tg: پلیمر = شیشه
        پیوندهای اسکلت انرژی حرارتی کافی
        برای حرکات پیچشی قابل توجه ندارند

بالای Tg: شروع حرکت پیچشی
          مولکول‌ها می‌توانند بپیچند و تسلیم تنش شوند
          پلیمر = شبه مایع (الاستومر)
          مگر با میکروکریستالیت سفت شود
```

**نتیجه‌گیری:**
```
پلیمر با Tg بالا → مقاومت بیشتر در برابر پیچش پیوند
پلیمر با Tg پایین → مقاومت کمتر در برابر پیچش پیوند
```

### **ویژگی غیرمعمول فسفازن‌ها** (صفحه ۱۴۷):

**پلی‌فسفازن‌ها دارای برخی از پایین‌ترین Tg های شناخته شده در شیمی پلیمر هستند:**
```
(NPCl₂)ₙ                              Tg = -66°C
(NPF₂)ₙ                               Tg = -96°C
[NP(OCH₃)₂]ₙ                          Tg = -74°C
[NP(OC₂H₅)₂]ₙ                         Tg = -84°C
[NP(OC₃H₇)₂]ₙ                         Tg = -100°C
[NP(OCH₂CH₂OCH₂CH₂OCH₃)₂]ₙ (MEEP)    Tg = -84°C
```

**توجه کنید:** مقادیر Tg پایین برای پلی‌فسفازن‌هایی است که:
- گروه‌های جانبی **بسیار کوچک** دارند (F، Cl، OCH₃)
- **یا** گروه‌های جانبی خودشان بسیار **انعطاف‌پذیر** هستند

### **اثر گروه‌های بزرگ/سخت** (صفحه ۱۴۸):

**اگر R ها بزرگ و سخت باشند:**
```
تداخل استریک با یکدیگر
وقتی پیوندهای اسکلت سعی در پیچش دارند
    ↓
قاعده کلی:
R های بزرگ و سخت → محدودیت بر انعطاف
حتی اگر پیوندهای اسکلت مانع کمی داشته باشند
```

**مثال‌ها:**
```
R = OPh                    Tg = -8°C
R = -OC₆H₄C₆H₅-p          Tg = +93°C
R = adamantylamino +       Tg = +180°C
    OCH₂CF₃ (mixed)
```

### **تطبیق‌پذیری سیستم** (صفحه ۱۴۹):

**با انتخاب R مناسب، پلیمرها را می‌توان طراحی کرد که:**
```
- الاستومرهای دمای پایین
- مواد انعطاف‌پذیر تشکیل‌دهنده فیلم و فیبر میکروکریستالین
- شیشه‌های ذوب بالا
```

**انتخاب R همچنین تأثیر می‌گذارد روی:**
- حلالیت
- ضریب شکست
- پایداری شیمیایی
- آبگریزی/آبدوستی
- رسانایی الکتریکی
- فعالیت نوری غیرخطی
- رفتار زیستی

# 📊 بخش ششم: روابط ساختار-خاصیت (صفحات ۱۵۰-۱۶۴)

## ۶.۱ تأثیر معماری مولکولی (صفحه ۱۵۱)

**مثال:** پلیمرهای خطی vs ستاره‌ای سه‌شاخه با OCH₂CF₃
```
همان وزن مولکولی (≥1.2×10⁴):

خطی:     مواد فیبری سفید
          فیلم و فیبر به راحتی
          کریستالین

ستاره:   صمغ‌های ویسکوز
          آمورف
```

**قاعده کلی:**
```
Cyclolinear:  معمولاً محلول و انعطاف‌پذیر
Cyclomatrix:  نامحلول و سخت
خطی:         می‌تواند کریستالین باشد
graft/comb:  معمولاً آمورف
```

---

## ۶.۲ کریستالین vs آمورف (صفحات ۱۵۲-۱۵۳)

### **اثر میکروکریستالیت** (صفحه ۱۵۲):
```
حوزه‌های میکروکریستالین در ماتریس آمورف:
- سفت کردن ماده
- ایجاد کدری (به جای شفافیت)
- افزایش دمای جریان مایع‌مانند
```

### **علت کریستالیزاسیون:**
```
تقارن مولکولی
    ↓
توالی منظم R ها در طول زنجیر
    ↓
بسته‌بندی فشرده با مولکول‌های همسایه
```

### **شرایط میکروکریستالینیتی در فسفازن‌ها** (صفحه ۱۵۳):

**فقط در یک مجموعه شرایط:**
```
1. فقط یک نوع R
2. R ها کوچک یا نسبتاً سخت

مثال‌های میکروکریستالین:
F, Cl, CH₃, C₃H₇, OCH₂CF₃, OC₆H₅
و فنوکسی‌های جانشین‌دار مختلف
```

**دو یا چند R مختلف:**
```
معمولاً فاقد تقارن و نظم لازم
    ↓
بیشتر پلی‌فسفازن‌های با جانشین‌های مختلط
    ↓
آمورف
```

**چرا؟**
روش جایگزینی → کنترل کافی روی استریوریگولاریتی ندارد

**استثنا:**
بیشتر پلی(آمینوفسفازن)ها آمورف هستند (شاید به خاطر پیوند هیدروژنی درون/بین مولکولی)

## ۶.۳ تغییرات مورفولوژی حالت جامد (صفحات ۱۵۴-۱۵۷)

### **گذارهای مزومورفیک** (صفحه ۱۵۶):

**علاوه بر تغییرات معمولی نزدیک Tg و Tm:**
```
برخی پلی‌فسفازن‌ها گذارهای مورفولوژیک
در دماهای بین Tg و Tm دارند
    ↓
گذارهای مزومورفیک یا کریستال مایع
(T₁, T₂, T₃, ...)
    ↓
تغییرات در کانفورماسیون یا بسته‌بندی زنجیر
```

**مثال:** `[NP(OCH₂CF₃)₂]ₙ` (شکل ۳.۱۱، صفحه ۱۵۷)

**اهمیت عملی:**
```
1. ساخت حرارتی در دماهای <Tm ممکن است
2. استحکام فیبر/فیلم ممکن است در دماهای
   خیلی پایین‌تر از Tm به شدت کاهش یابد
```

## ۶.۴ آبگریز vs آبدوست (صفحات ۱۵۸-۱۶۰)

### **اسکلت فسفازن** (صفحه ۱۵۸):
```
اسکلت خود آبدوست است
دلیل: الکترون‌های جفت‌نشده N
       توانایی تشکیل پیوند H با H₂O

اما
خاصیت کلی تعیین می‌شود توسط:
- گروه‌های جانبی
- میزان پوشش اسکلت توسط R ها
```

### **R های آبدوست** (صفحه ۱۵۹):
```
مثال‌ها:
-NHCH₃
-OCH₂CH₂OCH₂CH₂OCH₃
گلیکوسیل
گلیسریل
    ↓
حلالیت پلیمر در آب
```

**گروه‌های کوچک:**
```
CH₃: به اندازه کافی کوچک که N های اسکلت
     را پوشش نمی‌دهد
    ↓
پلیمرهای متیل‌فسفازن آبدوست
(اما حل در آب نمی‌شوند)
```

### **R های آبگریز** (صفحه ۱۵۹):
```
-OCH₂CF₃
-OC₆H₅
    ↓
آبگریز + به اندازه کافی بزرگ برای پوشش اسکلت
    ↓
آب‌گریزی قوی (شکل ۳.۱۲)
    +
حلالیت در حلال‌های آلی خاص
```

**مثال‌های حلالیت:**
```
[NP(OCH₲CF₃)₂]ₙ: محلول در استون، THF، اتیل استات
[NP(OPh)₂]ₙ: محلول در هیدروکربن‌های آروماتیک داغ
```

**اهمیت:**
- کاربردهای زیست‌پزشکی: تنظیم آبدوستی/آبگریزی حیاتی
- O-ringها در هواپیما/خودرو: عدم حلالیت در هیدروکربن‌ها مهم

## ۶.۵ پایدار در برابر آب vs قابل تخریب (صفحه ۱۶۱)

**تغییر نگرش:**
```
قبلاً: پلیمرهای ناپایدار در برابر آب
       → ارزش بیشتری ندارند

امروز: چنین موادی ارزش عظیمی دارند
        برای کاربردهای زیست‌پزشکی
        به خصوص اگر محصولات هیدرولیز غیرسمی باشند
```

**پلی‌فسفازن‌ها:**
```
بیشتر: پایدار در برابر آب ✅

تعداد انگشت‌شماری: ناپایدار ❌
شامل:
- پلیمرها با گروه‌های استر آمینواسید
- گونه‌هایی با ایمیدازولیل
- گونه‌هایی با گلیسریل
- گونه‌هایی با گلوکوسیل
```

## ۶.۷ ساختار مواد ناشی از چیدمان R (صفحات ۱۶۳-۱۶۴)

### **مفهوم** (صفحه ۱۶۳):

**در کریستال‌های مولکول کوچک:**
```
مولکول‌های بزرگ و صاف
    ↓
چیده می‌شوند مثل بشقاب‌ها
    ↓
نظم قابل توجه
    +
خواص نوری یا الکتریکی غیرمعمول
```

**در پلیمرها:**
```
R های بزرگ و صاف متصل به زنجیر انعطاف‌پذیر
    ↓
همان پدیده ممکن است رخ دهد
    ↓
گرایش R ها به تشکیل stack
ممکن است بر گرایش زنجیر غلبه کند (3.72)
```

**عمل:**
معمولاً یک "spacer انعطاف‌پذیر" لازم است برای جداسازی حرکات R ها از حرکات حرارتی زنجیر

### **پدیده‌ها** (صفحه ۱۶۴):

R های آروماتیک آزو یا بای‌فنیل
↓
کریستال مایعی

فتالوسیانین‌های فلزی + تترااسیانوکینودی‌متان
↓
حوزه‌های الکتروفعال

واحدهای چندآروماتیک (نفتیل، آنتراسن)
↓
تغییر خواص فیزیکی قابل توجه
وقتی محورهای مولکولی تراز می‌شوند
یا stack های R تشکیل می‌دهند

# 🏥 بخش هفتم: کاربردها (صفحات ۱۶۵-۲۵۲)

**این بخش بسیار گسترده است. به صورت خلاصه‌تر ارائه می‌شود اما

Continue

با جزئیات کلیدی برای امتحان**
۷.۱ الاستومرهای پیشرفته (صفحات ۱۶۵-۱۷۴)
سه نوع ساختار الاستومری (صفحه ۱۶۶):

نوع ۱: اتم‌های منفرد (F، Cl، Br):

اگر R = اتم‌های منفرد
    ↓
انعطاف ذاتی اسکلت غالب
    ↓
خاصیت الاستومری

اما مشکل:
(NPF₂)ₙ، (NPCl₂)ₙ، (NPBr₂)ₙ
    ↓
به آرامی در رطوبت هیدرولیز می‌شوند
    ↓
جالب به عنوان واسط واکنش
اما نه به عنوان مواد قابل استفاده
```

**نوع ۲: R های آلی انعطاف‌پذیر** (صفحه ۱۶۷):
```
R های انعطاف‌پذیر که به راحتی حرکات "اجتنابی"
انجام می‌دهند وقتی پیوندهای اسکلت می‌پیچند
    ↓
R ها مانع کم یا صفر برای پیچش اسکلت

مثال‌ها:
OCH₃، OC₂H₅، OC₃H₇، OC₄H₉
OCH₂CH₂OCH₃
-OCH₂CH₂OCH₂CH₂OCH₃
```

**مکانیسم کمکی:**
```
R ها خود کانفورماسیون‌های مختلف دارند
    ↓
فضا را به طور کارآمد پر نمی‌کنند
    ↓
حفره‌های مولکولی "حجم آزاد"
    ↓
فضا برای حرکت زنجیرها
    ↓
حجم آزاد بیشتر = انعطاف بیشتر
```

**پایداری:**
الاستومرها در این کلاس عموماً **پایدار در برابر آب** هستند

**نوع ۳: R های مختلط** (صفحات ۱۶۹-۱۷۴):

**اصل طراحی:**
```
توزیع random دو یا چند R مختلف
    ↓
کاهش گرایش به تراز بین زنجیری و کریستالینیتی
    +
تولید حجم آزاد (اگر R ها ابعاد مختلف)
    ↓
کمک به بازآرایی ماکرومولکولی
```

### **سه کلاس اصلی:**

**الف) فلوئورآلکوکسی‌های با جانشین‌های مختلط** (صفحات ۱۶۹-۱۷۱):

**اولین پلی‌فسفازن‌های تجاری:**
```
[NP(OCH₂CF₃)(OCH₂(CF₂)ₓCF₂H)]ₙ (3.73)

سنتز:
(NPCl₂)ₙ + مخلوط CF₃CH₂ONa + CHF₂(CF₂)ₓCH₂ONa
    ↓
رقابت برای جایگزینی Cl
    ↓
توزیع random + برخی واحدهای تک‌جانشین
```

**ساخت و کاربرد:**
```
معمولاً cross-linked در ساخت
    +
پرکننده‌ها (کربن سیاه، اکسید آهن)
    ↓
کاهش compliance (مقاومت بیشتر در برابر تغییر شکل)
```

**خواص و کاربردها:**
- ✅ انعطاف و کشسانی در دمای پایین (-60°C)
- ✅ مقاومت در برابر حلال‌های هیدروکربنی، روغن‌ها، مایعات هیدرولیکی
- ✅ مقاومت در برابر آتش

**کاربردها:**
- هوافضا
- خودروهای پیشرفته
- احتمالاً مواد زیستی بی‌اثر (به خاطر آبگریزی سطح)

**ب) آریلوکسی‌های با جانشین‌های مختلط** (صفحه ۱۷۲):
```
[NP(OPh)(OC₆H₄R)]ₙ (3.74)
R = آلکیل

Tg: نه به پایینی فلوئورآلکوکسی

کاربرد:
- غیرقابل اشتعال
- عایق صوتی و الکتریکی
- اغلب به صورت فوم لاستیک منبسط‌شده
```

**ج) ارگانوسیلیکون به عنوان cosubstituent** (صفحات ۱۷۳-۱۷۴):
```
[NP(CH₂SiMe₃)(OCH₂CF₃)]ₙ (3.75)
یا
[NP(OCH₂CF₃)₂]₂NP(Me)(CH₂SiMe₂OSiMe₃) (3.76)

سنتز:
1. پلیمریزاسیون تریمر با یک R ارگانوسیلیکون
2. جایگزینی Cl های باقیمانده با NaOCH₂CF₃
```

**ویژگی:**
هیبریدهای فسفازن و ارگانوسیلیکون

**علاقه:**
آبگریزی و خواص غشایی

## ۷.۲ غشاهای منفعل برای جداسازی (صفحات ۱۷۵-۱۸۳)

### **مفهوم غشا:**
```
فیلم‌های نازک پلیمری
    ↓
اجازه نفوذ سریع‌تر برخی مولکول‌ها نسبت به دیگران
```

**کاربرد:**
- جداسازی گازها
- جداسازی فاز مایع (دیالیز)

### **جداسازی گاز** (صفحه ۱۷۶):

**آریلوکسی‌های جایگزین‌شده:**
```
[NP(OPh)₂]ₙ غشاها:
- حذف SO₂ از N₂ در 30°C
- حذف CH₂Cl₂ یا CHCl₃ از هوا

[NP(OCH₂CF₃)₂]ₙ غشاها:
- نفوذپذیری بالا به CO₂، H₂، O₂
کاربردها:
- دستگاه تنفس زیر آب
- تکنولوژی پیل سوختی
```

### **جداسازی فاز مایع** (صفحات ۱۷۶-۱۸۳):

**غشاهای نیمه‌تراوا:**
```
اجزای ضروری تقریباً همه موجودات زنده
    +
توسعه مواد جدید = جزء مهم تحقیقات زیست‌پزشکی
```

**کنترل نفوذ:**
```
تغییرات در آبدوستی مولکول‌های پلیمر غشا
    ↓
مولکول‌های آبگریز → عبور آسان‌تر از حوزه‌های آبگریز
مولکول‌های آبدوست → عبور آسان‌تر از حوزه‌های آبدوست
```

**فرصت در فسفازن‌ها:**
```
تغییر نسبت R های آبدوست/آبگریز
    ↓
تولید طیفی از خواص "آمفی‌فیلیک"
    ↓
مفید در طراحی و تکنولوژی غشا
```

**مثال: پلیمرهای ۳.۷۷ و ۳.۷۸** (صفحات ۱۷۸-۱۷۹):
```
NHCH₃ = آبدوست
OCH₂CF₃ و OPh = آبگریز

با تغییر نسبت از 0-100%:
    ↓
تغییر خواص سطحی و رفتار غشایی

زاویه تماس:
[NP(NHCH₃)₂]ₙ: ≤30°
[NP(OCH₂CF₃)₂]ₙ یا [NP(OPh)₂]ₙ: 100-107°
```

**رفتار نیمه‌تراوایی:**
```
سرعت نفوذ رنگ متغیر با نسبت R

پلیمر با ~50% متیل‌آمینو + 50% تری‌فلوئورواتوکسی:
    ↓
انتقال سریع‌تر مولکول‌های رنگ در آب
نسبت به لوله دیالیز سلولز استاندارد
```

**بهینه‌سازی بیشتر:**

**مشکل:**
```
پلیمرهای با نسبت بالای NHCH₃:
- خیلی شکننده (Tg بالا)
- خیلی محلول در محیط آبی
```

**راه حل (صفحه ۱۸۲، واکنش ۵۴):**
```
Cross-linking تابشی

پلیمر محلول در آب --[تابش گاما/UV]-->
                      نامحلول اما متورم در آب
                      (هیدروژل)
```

## ۷.۳ غشاهای پاسخ‌دهنده (صفحات ۱۸۴-۱۸۷)

### **تعریف:**
```
غشایی که نفوذپذیری به املاح محلول
می‌تواند تغییر کند با تغییر در:
- دما
- pH
- قدرت یونی
- شدت نور
- جریان الکتریکی
- میدان مغناطیسی
```

**کاربرد اصلی:**
```
استفاده از غشا به عنوان "سوئیچ"
برای روشن/خاموش کردن جریان مولکول‌ها
در پاسخ به تغییرات محیطی
```

**مثال زیست‌پزشکی:**
```
رهش دارو از دستگاه کاشته‌شده
    ↓
کنترل خودکار در پاسخ به:
- تغییرات دما بیمار
- بیوشیمی بیمار
```

### **هیدروژل‌های پلی‌فسفازن** (صفحه ۱۸۵، واکنش ۵۵):

**اصل:**
```
فسفازن‌ها با زنجیرهای جانبی آلکیل اتر
مثل -OCH₂CH₂OCH₂CH₂OCH₃
    ↓
به راحتی cross-link با UV یا گاما
```

**خواص:**

**۱. LCST (Lower Critical Solution Temperature):**
```
زیر LCST: پلیمر محلول
بالای LCST: پلیمر رسوب می‌کند

هیدروژل‌های cross-linked:
زیر LCST: متورم با آب
بالای LCST: منقبض
```

**محدوده LCST در فسفازن‌ها:** 30-65°C

**جالب‌ترین‌ها:** LCST نزدیک 38°C (دمای بدن)

**۲. پاسخ به pH:**
```
Cosubstituent با واحدهای اسید کربوکسیلیک
مثل -OC₆H₄COOH
    ↓
هیدروژل پاسخ‌دهنده به pH

pH بالا: انبساط
pH پایین: انقباض
```

**عملکرد به عنوان غشا (صفحه ۱۸۷، شکل ۳.۱۵):**
```
غشا منقبض شده: جلوگیری از انتقال حلال و املاح
غشا منبسط شده: اجازه نفوذ
```

## ۷.۴ غشاها از رساناهای یونی جامد (صفحات ۱۸۸-۱۹۷)

### **دو نوع مهم:**

**۱. رساناهای کاتیون فلزی (مثل Li⁺):**
- برای الکترولیت در باتری‌های لیتیومی قابل شارژ پیشرفته

**۲. رساناهای پروتون:**
- اجزای حیاتی در پیل‌های سوختی

### **الف) رساناهای یون فلزی** (صفحات ۱۸۹-۱۹۴)

**پلیمرهای با الیگو-اتیلن‌اکسی:**
```
[NP(OCH₂CH₂OCH₂CH₂OCH₃)₂]ₙ (MEEP)
MethoxyEthoxyEthoxy Phosphazene

ویژگی‌ها:
- انعطاف مولکولی بالا
- توانایی coordination ضعیف به Li⁺ و سایر کاتیون‌های تک‌ظرفیتی
```

**عملکرد (صفحه ۱۹۰-۱۹۱، شکل ۳.۱۶):**
```
MEEP به عنوان حلال جامد برای نمک‌ها:
- LiSO₃CF₃ (لیتیوم تریفلات)
- Li trifluorosulfonimide
- AgSO₃CF₃

مکانیسم رسانایی:
1. solvation یون‌ها (به خصوص کاتیون‌ها)
   توسط اتم‌های O در R ها
   → جدایی جفت یون

2. زنجیرهای کافی انعطاف‌پذیر
   + حجم آزاد کافی
   → انتقال کاتیون‌ها از زنجیر به زنجیر
   (حرکات حرارتی ماکرومولکولی)

3. اعمال جریان الکتریکی:
   کاتیون‌ها → به سمت الکترود منفی
   آنیون‌ها → به سمت الکترود مثبت
```

**کاربرد: باتری لیتیومی** (صفحه ۱۹۲، شکل ۳.۱۷):
```
ساختار (مقطع):
┌─────────────────────┐
│   آند (Li فلز)     │
├─────────────────────┤
│ الکترولیت جامد     │
│  (MEEP + LiSO₃CF₃) │
├─────────────────────┤
│ کاتد (MnO₂ یا...)  │
└─────────────────────┘

مزیت:
تئوری هیچ محدودیتی برای اندازه
چون الکترولیت جامد قابل ریخته‌گری به صورت فیلم
```

**مقایسه با PEO:**
```
پلی(اتیلن اکساید) (PEO):
- پلیمر نمونه قبلی
- اما میکروکریستالین
- رسانایی یونی در نواحی آمورف
  قطع می‌شود توسط حوزه‌های میکروکریستالین
- باید تا ~100°C گرم شود برای رسانایی بالا

MEEP (3.79) و انواع آن:
- غیرکریستالین
- رسانایی یونی در دمای اتاق
  ≥1000 برابر PEO!

برای باتری: MEEP باید کمی cross-link شود
(جلوگیری از جریان آهسته مایع‌مانند)
    ↓
قابل انجام با تابش بدون کاهش رسانایی
```

**معماری‌های دیگر:**
```
- فسفازن‌ها با زنجیرهای جانبی آلکیل اتر منشعب
- ماکرومولکول‌های هیبرید:
  * زنجیر پلیمر آلی + حلقه‌های تریمر فسفازن جانبی (3.16)
    با واحدهای آلکیل اتر
  * زنجیرهای پلیمر آلی نوع comb (3.15)
    با زنجیرهای جانبی نوع MEEP
  * کوپلیمرهای block MEEP + PEO

بالاترین رسانایی دمای اتاق: ~10⁻⁴ S/cm
(تقریباً یک مرتبه کمتر از مقدار لازم برای بسیاری کاربردها)
```

**الکترولیت‌های ژل:**
```
رسانایی‌ها با افزودن مقادیر کم
مایعات آلی هماهنگ‌کننده (مثل پروپیلن کربنات):
    ↓
افزایش به محدوده 10⁻³

مزیت نسبت به PEO متورم با حلال:
مقدار حلال آلی قابل اشتعال لازم **بسیار کمتر**
```

### **ب) رساناهای پروتون** (صفحات ۱۹۵-۱۹۷)

**ساختار پیل سوختی الکترولیت پلیمری** (صفحه ۱۹۶، شکل ۳.۱۸):
```
آند: سوخت (H₂ یا متانول) → e⁻ + H⁺
      e⁻ → مدار خارجی → بار → کاتد
      H⁺ → غشای پلیمری → کاتد

کاتد: H⁺ + O₂ + e⁻ → H₂O
```

**الزامات:**
- پایداری شیمیایی و حرارتی
- رسانایی پروتون
- **مقاومت در برابر انتقال سوخت** به کاتد

**مشکل غشاهای تجاری (مثل Nafion):**
```
متانول به راحتی از Nafion عبور می‌کند
    ↓
یکی از موانع اصلی توسعه
پیل‌های سوختی متانول مستقیم
```

**فسفازن‌های رسانای پروتون** (صفحه ۱۹۷، ۳.۸۰-۳.۸۲):
```
3.80: [NP(OC₆H₄SO₃H)₂]ₙ
3.81: [NP(OC₆H₄COOH)₂]ₙ  
3.82: [NP(OC₆H₄SO₂NHSO₂CF₃)₂]ₙ (sulfonimide)

همه: آریلوکسی‌فسفازن‌ها با گروه‌های عاملی اسیدی روی حلقه‌های آریل
```

**عملکرد:**
```
قابل مقایسه با Nafion در پیل‌های سوختی عملیاتی
اما
مقاوم‌تر در برابر عبور متانول

مشتق sulfonimide (3.82):
رسانایی پروتون در دماهای بالاتر از Nafion
    ↓
مزیت برای توسعه‌های آینده
```

## ۷.۵ فیبرها، فیلم‌ها و سطوح ویژه (صفحات ۱۹۸-۲۰۰)

### **الزامات فیبر خوب:**
```
(a) Tg زیر دمای اتاق (انعطاف در دماهای عادی)
(b) میکروکریستالینیتی (استحکام، قابلیت جهت‌گیری)
```

**فسفازن‌های میکروکریستالین:**
```
معمولاً فقط با پلیمرهای تک‌جانشین
مثل: OCH₃CF₃، OPh، OC₆H₄R-p
(اگرچه تعداد کمی از مشتقات با جانشین‌های مختلط هم کریستالین هستند)

بسیاری = تشکیل‌دهنده فیلم و فیبر
```

**ساخت:**
```
راحت‌تر با:
- solution-casting
- solution-spinning

نه melt-fabrication

چرا؟
میکروکریستالیت‌ها در دماهای بسیار بالا ذوب می‌شوند
(گاهی >250°C)
    ↓
پلیمرها شروع به واپلیمریزاسیون می‌کنند
```

### **فیلم‌ها و پوشش‌های سطحی:**
```
بسیاری از فسفازن‌ها فیلم‌های قوی تشکیل می‌دهند
روش: گسترش محلول روی سطح صاف
      + تبخیر حلال

برخی فیلم‌ها خواص ویژه دارند:
- مقاومت در برابر تابش UV
    ↓
کاربرد: پوشش برای سلول‌های خورشیدی، هواپیما، سفینه فضایی

[NP(OCH₂CF₃)₲]ₙ و مشتقات با جانشین‌های مختلط:
مطالعه شده از این دیدگاه
```

**پوشش‌های سطحی:**
```
نیاز به:
- چسبندگی قوی به سطح
- cross-linking پلیمر

پلیمرهای Cyclolinear و cyclomatrix:
به شدت بررسی شده برای این کاربردها
```

### **فیلم‌ها با سطوح ویژه:**

**مثال: تولید سطح چسبنده** (واکنش ۲۹):
```
[NP(OCH₂CF₃)₂]ₙ فیلم + NaOH/Bu₄NBr (آبی)
    ↓
سطح چسبنده روی یک یا هر دو طرف

فراهم می‌کند وسیله‌ای برای چسباندن
فیلم‌های محافظ این پلیمر به مواد دیگر
```

**مثال دیگر: تولید سطح غیرچسبنده:**
```
[NP(OCH₂CF₃)(OCH₂(CF₂)CF₂H)]ₙ الاستومر چسبنده
    + NaOCH₲(CF₂)CF₂F
    ↓
سطح غیرچسبنده

ادامه در بخش بعدی به دلیل طول زیاد...

نکات مهم برای امتحان از بخش کاربردها:

    الاستومرها: سه نوع، خواص هر کدام، کاربردها
    غشاها: منفعل (جداسازی)، پاسخ‌دهنده (سوئیچ)، رساناهای یونی (باتری/پیل سوختی)
    MEEP: ساختار، رسانایی یونی، مزیت نسبت به PEO
    زیست‌پزشکی: (بخش بعدی - بسیار مهم!)

آیا می‌خواهید ادامه بخش کاربردها (زیست‌پزشکی) را به همین تفصیل ببینید؟
